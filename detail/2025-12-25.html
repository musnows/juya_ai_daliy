1. 🚀 TRAE中国版开放SOLO模式并内置多款免费国产模型

标签： TRAE 豆包 GLM Kimi

TRAE中国版宣布逐步向全部用户免费开放SOLO模式，用户可自主选择模型进行独立开发。该模式内置豆包、GLM Mini Max、Kimi等6款最新国产免费模型，支持开发者根据需求灵活切换底层大模型。此次更新显著降低了AI开发门槛，尤其利好中小企业和个人开发者。SOLO模式强调轻量化与自主性，适用于原型开发、代码生成和本地调试等场景。结合国产模型生态，进一步推动本土化AI工具链建设。

2. 🔧 Windsurf发布Wave 13更新，上线SWE-1.5 Free模型

标签： Windsurf SWE-1.5 Free Cognition

Cognition公司发布Windsurf Wave 13版本更新，为所有用户提供为期3个月的SWE-1.5 Free模型免费使用权限。新版本增强了对Git工作流的支持，实现多窗口并行Agents协作，提升开发效率。同时推出macOS交互式终端测试版，支持实时命令执行与反馈。SWE-1.5 Free基于Qwen3-Coder微调，专为软件工程任务优化，在代码补全、重构和调试方面表现优异。

🔗 相关链接：
- https://windsurf.com/blog/windsurf-wave-13

6. 🔧 智谱AI上线Zread MCP服务器支持IDE内分析GitHub仓库

标签： 智谱AI GLM Coding Plan Zread MCP

智谱AI推出专为GLM Coding Plan用户设计的Zread MCP服务器，允许开发者在IDE内通过Agent直接分析GitHub开源仓库。调用额度根据用户订阅套餐等级确定，高级用户享有更高频次与深度分析权限。该工具支持代码结构解析、依赖关系追踪与漏洞检测，集成于主流开发环境如VS Code，提升开源项目研究效率。

7. 🚀 Kilo Code发布App Builder浏览器端应用开发工具

标签： Kilo Code App Builder

Kilo Code推出App Builder工具，支持用户通过自然语言对话实时构建并预览网页项目。该工具完全基于浏览器运行，无需本地安装，集成拖拽式界面与AI代码生成能力。用户可描述功能需求，系统自动生成HTML、CSS与JavaScript代码，并支持实时调试与导出。适用于快速原型设计与非技术用户入门开发。

🔗 相关链接：
- https://blog.kilo.ai/p/introducing-app-builder
- https://app.kilo.ai/app-builder

8. 🔧 StepFun发布NextStep-1.1图像生成模型

标签： StepFun NextStep-1.1

StepFun发布150亿参数图像生成模型NextStep-1.1，通过强化学习技术大幅提升了图像质量与高保真输出稳定性。该模型在细节还原、色彩一致性与复杂场景生成方面表现突出，支持文本到图像、图像修复与风格迁移等任务。已在Hugging Face开源，并提供API接口供开发者集成使用。

🔗 相关链接：
- https://huggingface.co/stepfun-ai/NextStep-1.1
- https://github.com/stepfun-ai/NextStep-1

9. 🔧 Nebius发布OpenHands轨迹数据集与RFT微调模型

标签： Nebius OpenHands Trajectories Qwen3-Coder

Nebius发布包含6.7万条Agent轨迹的大规模数据集OpenHands Trajectories，用于支持软件工程AI研究。同步推出两款基于Qwen3-Coder-480B的微调模型，采用RFT（Reinforced Fine-Tuning）方法训练，在SWE-bench等基准测试中表现优异。数据集涵盖真实开发场景中的代码修改、调试与协作行为，助力Agent智能体能力提升。

🔗 相关链接：
- https://nebius.com/blog/posts/openhands-trajectories-with-qwen3-coder-480b
- https://huggingface.co/datasets/nebius/SWE-rebench-openhands-trajectories

12. 📈 英伟达与Groq达成200亿美元推理技术许可协议
