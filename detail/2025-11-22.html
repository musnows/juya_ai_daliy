1. 🚀 腾讯开源HunyuanVideo-1.5视频模型

标签： 腾讯 HunyuanVideo-1.5 Hugging Face

腾讯于近日正式开源其新一代视频生成模型 HunyuanVideo-1.5，该模型在 Hugging Face、GitHub 及腾讯云官网同步上线，提供完整的模型权重、推理代码与文档支持。HunyuanVideo-1.5 是 HunyuanVideo 系列的升级版本，具备更强的视频生成质量、更高的分辨率支持以及更优的语义对齐能力，支持文本到视频（Text-to-Video）和图像到视频（Image-to-Video）生成任务。模型基于大规模视频-文本数据集训练，采用先进的扩散架构与时空建模技术，显著提升了生成视频的时序连贯性与细节表现力。此次开源进一步推动了视频生成技术的社区发展，为学术研究、内容创作及工业应用提供了高性能、可复现的基础模型。用户可通过 Hugging Face 模型库、GitHub 仓库及腾讯云 Hunyuan 官网获取模型资源。

🔗 相关链接：
- https://huggingface.co/tencent/HunyuanVideo-1.5
- https://hunyuan.tencent.com/video/zh?tabIndex=0
- https://github.com/Tencent-Hunyuan/HunyuanVideo-1.5

2. 🚀 小米发布跨具身基础模型MiMo-Embodied

标签： 小米 MiMo-Embodied 具身智能

小米发布跨具身基础模型 MiMo-Embodied，该模型以 7B 参数规模在 Hugging Face 平台开源，论文已上传至 arXiv（编号 2511.16518）。MiMo-Embodied 旨在实现机器人与物理环境的跨模态感知与交互能力，支持视觉、语言与动作的联合建模，具备在复杂场景中理解指令、规划动作并执行任务的能力。模型采用多模态预训练架构，融合图像、文本与传感器数据，适用于家庭服务机器人、工业自动化等具身智能（Embodied AI）场景。其开源将促进具身智能领域的算法研究与落地应用，推动机器人从‘感知’向‘认知+执行’演进。

🔗 相关链接：
- https://arxiv.org/abs/2511.16518
- https://huggingface.co/XiaomiMiMo/MiMo-Embodied-7B

3. 🚀 ServiceNow-AI发布Apriel-H1模型系列

标签： ServiceNow-AI Apriel-H1 企业级AI

ServiceNow-AI 正式发布 Apriel-H1 模型系列，该系列模型聚焦于企业级 AI 应用，支持代码生成、文档理解与自动化流程推理等任务。Apriel-H1 系列在 Hugging Face 上建立专属模型集合（collection），并发布官方博客详细介绍其训练方法、性能基准与部署方案。模型针对企业 IT 运维、客户服务与流程自动化场景优化，具备高准确率、低延迟与强安全性，支持私有部署与 API 调用。Apriel-H1 的发布标志着 ServiceNow 在 AI 原生企业服务领域的进一步布局，有望提升企业数字化转型的智能化水平。

4. 🚀 豆包输入法上线多语种语音输入功能

标签： 豆包输入法 多语种语音输入 字节跳动

字节跳动旗下豆包输入法正式上线多语种语音输入功能，支持中文、英文、日语、韩语、法语、西班牙语等十余种语言的实时语音转文字。该功能依托豆包大模型在语音识别（ASR）与多语言理解方面的能力，具备高识别准确率、低延迟与离线模式支持，适用于会议记录、跨语言沟通、内容创作等场景。用户可通过官网 shurufa.doubao.com 下载体验。此次更新标志着豆包输入法从单一中文输入向全球化、多模态输入工具的转型，进一步拓展其在智能输入领域的竞争力。

🔗 相关链接：
- https://shurufa.doubao.com/
