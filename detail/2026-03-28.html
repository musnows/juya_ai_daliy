1. 🚀 智谱向GLM Coding Plan用户开放GLM-5.1模型使用

标签： 智谱 GLM-5.1 GLM Coding Plan

智谱AI于2026年3月28日正式向GLM Coding Plan订阅用户开放新一代代码生成模型GLM-5.1的使用权限。该模型作为智谱在编程智能领域的最新成果，具备更强的代码理解、生成与调试能力，支持多语言编程任务，并在复杂逻辑推理和长上下文处理方面实现显著提升。此次开放标志着智谱进一步推动AI编程助手在企业级开发场景中的落地应用，有望提升开发者效率并降低编码门槛。GLM-5.1的推出也体现了智谱在垂直领域大模型持续迭代的技术路线。

2. 🚀 KwaiKAT发布KAT-Coder-Pro V2代码模型

标签： KwaiKAT KAT-Coder-Pro V2

快手旗下KwaiKAT团队正式发布KAT-Coder-Pro V2模型，该版本在代码生成准确率、多语言支持及上下文理解能力上较前代有显著优化。据官方介绍，V2版本在HumanEval等主流代码评测基准上表现优异，特别强化了对Python、Java和JavaScript等主流编程语言的支持，适用于自动化编程辅助、代码补全与重构等场景。该模型的发布进一步丰富了国内AI编程工具生态，助力开发者提升工作效率。

3. 🔧 Meta发布SAM 3.1，支持单次追踪16个目标

标签： Meta SAM 3.1

Meta公司发布视觉分割模型SAM（Segment Anything Model）的3.1版本，新增支持单次推理同时追踪多达16个目标对象的能力。该升级显著提升了模型在复杂场景下的实例分割效率，适用于视频分析、自动驾驶、医学影像处理等对多目标识别要求较高的领域。SAM 3.1在保持高精度分割的同时，优化了计算资源消耗，增强了实时处理能力，进一步巩固了其在通用视觉基础模型中的领先地位。

4. 🔧 Claude Code上线Windows PowerShell原生运行支持

标签： Anthropic Claude Code

Anthropic宣布其AI编程助手Claude Code现已支持在Windows系统的PowerShell环境中原生运行。此举打破了此前主要依赖Linux或macOS终端的限制，极大拓展了Windows开发者的使用场景。用户可直接在PowerShell中调用Claude Code进行代码生成、调试与解释，实现无缝集成。该更新提升了跨平台开发体验，有助于推动Claude Code在企业级开发环境中的普及。

5. 📈 OpenRouter限制多地用户调用部分模型，称遵上游要求

标签： OpenRouter

AI模型路由平台OpenRouter宣布对部分地区的用户实施模型调用限制，涉及多个主流大模型服务。平台表示该措施系遵循上游模型提供方的合规要求，具体受限区域和模型未完全公开。此举可能影响依赖OpenRouter进行多模型调用的开发者和企业用户，反映出AI服务在全球化部署中面临的地缘合规挑战。

6. 📈 OpenAI重置Codex限额，调整开发者访问策略

标签： OpenAI Codex

OpenAI宣布对Codex模型的API调用限额进行重置，并调整相关访问策略。此次调整可能涉及免费层与付费层的配额重新分配，旨在优化资源使用并提升服务稳定性。Codex作为早期广泛应用的代码生成模型，其政策变动对依赖该API的开发者社区具有一定影响，也反映出OpenAI在模型服务管理上的持续优化。

7. 🔧 NotebookLM支持后台生成与移动端推送通知

标签： Google NotebookLM

Google旗下AI笔记工具NotebookLM推出新功能，支持在后台持续生成内容摘要与问答，并向移动端设备推送实时通知。用户可在不打开应用的情况下接收关键信息更新，提升知识管理效率。该功能特别适用于研究人员和学习者，能够在多文档分析场景中实现更智能的信息提炼与提醒，增强产品的实用性与用户粘性。

8. 🚀 钉钉开源dws跨平台命令行工具，面向AI Agent场景

标签： 钉钉 dws

钉钉正式开源名为dws的跨平台命令行工具，专为AI Agent开发场景设计。该工具支持在Windows、macOS和Linux系统上运行，提供统一的接口用于任务调度、环境配置与Agent交互。dws的开源有助于降低AI Agent开发门槛，促进企业内部自动化流程的构建，体现了钉钉在智能办公与AI融合方向的技术投入。

9. 🚀 硅基流动推出AI算力运营服务

标签： 硅基流动

硅基流动公司正式推出AI算力运营服务，旨在为企业和研究机构提供高效、灵活的算力资源管理与调度解决方案。该服务支持多云环境下的算力整合、成本优化与任务分发，适用于大模型训练与推理等高计算需求场景。此举标志着硅基流动从技术平台向算力基础设施服务商的战略延伸，有望推动AI算力的普惠化应用。

10. 📈 Anthropic胜诉，拟四季度筹资600亿美元上市

标签： Anthropic

Anthropic在一项关键法律诉讼中胜诉，扫清了其上市进程中的重要障碍。公司计划在2026年第四季度进行首次公开募股（IPO），拟筹资规模达600亿美元，估值有望刷新AI领域纪录。此次胜诉增强了投资者信心，若成功上市，Anthropic将成为全球市值最高的AI初创企业之一，进一步加剧与大模型头部厂商的竞争格局。

11. 📈 OpenAI ChatGPT广告年化收入突破1亿美元

标签： OpenAI ChatGPT

据最新数据显示，OpenAI旗下ChatGPT通过广告模式实现的年化收入已突破1亿美元。该收入主要来源于企业合作、品牌推广及定向广告投放等商业化尝试。尽管ChatGPT仍以订阅服务为主要盈利模式，但广告收入的快速增长表明其在B端营销与用户触达方面具备巨大潜力，也为AI产品的多元化变现提供了新路径。

12. 📈 Anthropic泄露未发布模型Claude Mythos

Anthropic公司确认，其尚未正式发布的内部模型Claude Mythos发生数据泄露事件。该模型原计划作为Claude系列的高端版本推出，具备更强的推理与多模态能力。泄露事件可能影响产品发布节奏与安全策略，公司已启动内部调查并加强模型访问管控。此次事件再次引发对AI公司核心模型安全管理的关注。
