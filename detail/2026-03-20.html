1. 🚀 Cursor 发布 Composer 2 模型，强化学习提升长周期编程能力

标签： Cursor Composer 2

Cursor 正式发布 Composer 2 模型，该版本通过引入强化学习技术，显著提升了在长时间、多步骤编程任务中的表现。Composer 2 能够更好地理解复杂项目上下文，支持跨文件编辑与逻辑连贯的代码生成，适用于大型代码库的迭代开发。此次升级标志着 AI 编程助手在自主性与任务持续性方面的重要进展，有望提高开发者效率并降低维护成本。

2. 🔧 Claude Code 推出 Channels 功能，支持手机远程操控

标签： Claude Code Channels

Claude Code 新增 Channels 功能，允许用户通过移动设备远程访问和控制本地开发环境。该功能实现了跨终端的实时协作与代码调试，开发者可在外出时通过手机查看日志、执行命令或接收任务通知。Channels 基于安全加密通道构建，确保远程操作的安全性，进一步拓展了 AI 编程工具的移动办公场景。

3. 🚀 Google AI Studio 上线全栈 Vibe Coding 功能

标签： Google AI Studio Vibe Coding

Google AI Studio 正式推出全栈 Vibe Coding 功能，支持从前端界面到后端逻辑的端到端代码生成。该功能结合自然语言描述与上下文感知技术，可自动生成完整应用模块，包括 API 接口、数据库模型与用户界面。Vibe Coding 面向全栈开发者，旨在降低多技术栈协同开发的复杂度，提升原型构建速度。

4. 🚀 微软 AI 发布 MAI-Image-2 文生图模型

标签： 微软 AI MAI-Image-2

微软 AI 部门发布新一代文生图模型 MAI-Image-2，具备更高的图像分辨率与语义理解精度。该模型支持复杂提示词解析，可生成符合工业设计标准的图像内容，适用于广告、UI 设计与内容创作等领域。MAI-Image-2 在细节还原与风格一致性方面表现突出，已集成至微软 Copilot 生态系统中。

5. 🔧 小红书 rednote-hilab 发布 dots.mocr，3B 参数支持图像转 SVG

标签： 小红书 rednote-hilab dots.mocr

小红书旗下 rednote-hilab 实验室发布轻量级模型 dots.mocr，参数规模为 30 亿，专注于将位图图像转换为可缩放的 SVG 矢量图形。该模型在保持图形结构完整性的同时显著压缩文件大小，适用于网页图标、UI 元素与插画资源的自动化处理，提升前端开发效率。

6. 🔧 TRAE IDE 登陆 Linux 平台，实现三大系统兼容

标签： TRAE IDE

TRAE IDE 正式支持 Linux 操作系统，至此已完成对 Windows、macOS 和 Linux 三大主流平台的全面兼容。此举扩大了其在开源社区与服务器端开发场景中的应用范围，支持跨平台项目协作与容器化部署，增强了开发环境的灵活性与可移植性。

7. 📈 Gemini API 更新计费策略，2026年4月设支出上限

标签： Gemini API

Google 宣布自 2026 年 4 月起，Gemini API 将引入支出上限机制，防止因异常调用导致的高额费用。新计费策略允许开发者设置月度预算阈值，超出后将自动暂停服务，提升成本可控性。该更新面向企业级用户，旨在增强 API 使用的安全性与可预测性。

8. 📈 Windsurf 启用新定价，推出200美元Max计划

标签： Windsurf

Windsurf 调整产品定价结构，新增每月 200 美元的 Max 计划，提供无限制 AI 辅助、优先支持与高级协作功能。新定价体系旨在满足专业开发者与团队的高强度使用需求，同时保留基础免费版以吸引新用户。

9. 🔧 Cursor 发布早期 Alpha 版界面 Glass

标签： Cursor Glass

Cursor 推出实验性用户界面 Glass，采用透明层叠设计与沉浸式交互体验。该 Alpha 版本聚焦于减少视觉干扰，提升代码编辑专注度，支持动态信息浮层与上下文感知提示。Glass 界面目前处于早期测试阶段，未来或将成为默认 UI 选项。

10. 🔧 LlamaIndex 开源 LiteParse，无需GPU秒解析五百页文档

标签： LlamaIndex LiteParse

LlamaIndex 开源轻量级文档解析工具 LiteParse，可在无 GPU 环境下每秒处理超过 500 页 PDF 或扫描文档。该工具采用高效文本提取与结构识别算法，支持表格、公式与多栏布局的准确还原，适用于大规模知识库构建与 RAG 应用。

11. 🚀 字节小云雀 AI 上线短剧 Agent，全流程自动制作

标签： 字节跳动 小云雀 AI

字节跳动旗下小云雀 AI 推出短剧 Agent，实现从剧本生成、分镜设计到视频合成的全流程自动化制作。该 Agent 基于多模态大模型驱动，可依据用户输入的主题快速产出符合平台规范的短视频内容，显著降低短剧创作门槛与成本。

12. 🚀 阶跃星辰正式上线本地 AI Agent StepClaw

标签： 阶跃星辰 StepClaw

阶跃星辰正式推出本地部署的 AI Agent 产品 StepClaw，支持在私有环境中运行复杂任务自动化流程。StepClaw 具备本地数据处理能力，保障企业敏感信息安全，适用于金融、医疗等对数据合规要求较高的行业场景。

13. 📈 OpenAI 披露内部 Agent 监控，覆盖99.9%流量未现高风险

标签： OpenAI

OpenAI 公开其内部 Agent 系统的监控机制，表示已实现对 99.9% 流量的实时行为分析，目前未发现高风险异常操作。该监控系统用于检测恶意提示、越权访问与模型滥用行为，是保障 AI 系统安全运行的关键基础设施。

14. 📈 OpenAI 宣布收购 Astral，整合开源工具至 Codex 生态

标签： OpenAI Astral Codex

OpenAI 宣布收购开源开发工具公司 Astral，计划将其旗下工具链整合至 Codex 生态系统。Astral 专注于开发者效率工具，此次收购将增强 OpenAI 在代码补全、调试辅助与团队协作方面的能力，进一步巩固其在 AI 编程领域的领先地位。

15. 📈 MiniMax 对 M2.7 实行高峰时段限流，新购用户增加周限

标签： MiniMax M2.7

MiniMax 宣布对 M2.7 模型在高峰时段实施限流措施，以保障服务稳定性。同时，新购买用户将面临每周调用次数上限，现有企业客户可申请配额调整。该政策反映大模型服务在高并发场景下的资源管理挑战。

16. 📈 昆仑万维天工 SkyReels V4 获 Artificial Analysis 视频榜第一

标签： 昆仑万维 SkyReels V4

昆仑万维旗下天工 SkyReels V4 模型在 Artificial Analysis 发布的视频生成榜单中位列第一，凭借其在视频连贯性、画质清晰度与动作自然度方面的优异表现获得认可。该模型支持长视频生成与多镜头调度，适用于影视预览与内容创作。

17. 🚀 阿里千问官宣 Qwen3.5-Max-Preview 模型亮相 LM Arena

标签： 阿里巴巴 Qwen3.5-Max-Preview

阿里巴巴千问团队正式推出 Qwen3.5-Max-Preview 模型，并在 LM Arena 平台开放测试。该模型在多项基准测试中表现优异，尤其在中文理解与代码生成任务上具备竞争力，标志着阿里在大模型技术上的持续突破。

18. 📈 腾讯确认研发微信 AI 智能体，或于第三季度上线

标签： 腾讯 微信 AI 智能体

腾讯官方确认正在研发基于微信平台的 AI 智能体，预计将于 2026 年第三季度正式上线。该智能体将深度集成于微信生态，支持消息处理、服务调用与个性化推荐，有望重塑社交场景下的 AI 交互体验。
