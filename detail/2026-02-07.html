1. 🚀 美团发布LongCat-Flash-Lite模型，提供免费API服务

标签： 美团 LongCat-Flash-Lite SGLang-FluentLLM

美团于2026年2月7日正式推出轻量级大语言模型LongCat-Flash-Lite，并开放免费API接口供开发者调用。该模型基于SGLang-FluentLLM架构优化，具备高效推理能力和低延迟响应特性，适用于对话生成、文本摘要及代码辅助等场景。用户可通过Hugging Face平台获取模型权重，GitHub仓库提供完整部署方案。此举标志着美团在开源AI生态布局上的进一步深入，有望降低中小企业接入先进语言模型的技术门槛。

2. 🚀 字节跳动发布视频生成模型Seedance 2.0

标签： 字节跳动 Seedance 2.0

字节跳动于2026年2月7日发布新一代视频生成模型Seedance 2.0，该版本在视频连贯性、分辨率和语义理解方面实现显著提升。据官方文档显示，Seedance 2.0支持多模态输入（文本+图像），可生成长达60秒的高质量短视频，适用于广告创意、内容营销和虚拟主播等应用场景。模型已在字节内部多个产品线完成集成测试，未来或向B端客户开放API服务。

3. 🔧 OpenRouter与Kilo Code联合上线Pony Alpha神秘模型

标签： OpenRouter Kilo Code Pony Alpha

OpenRouter平台与Kilo Code于2026年2月7日共同推出代号为Pony Alpha的未公开模型，该模型具备“深度思考”能力，支持复杂逻辑推理与长上下文处理。据Kilo Code博客介绍，Pony Alpha采用新型稀疏注意力机制，在数学证明和代码生成任务中表现突出。目前该模型仅通过OpenRouter平台限量开放，暂未披露训练数据来源与参数量级，引发业界对其技术路线的广泛猜测。

4. 🔧 Waymo发布自动驾驶仿真系统Waymo World Model

标签： Waymo Waymo World Model

Waymo于2026年2月7日推出全新自动驾驶仿真平台Waymo World Model，旨在构建高保真虚拟驾驶环境以加速算法训练与验证。该系统融合真实道路数据与生成式建模技术，可模拟极端天气、罕见交通事件等长尾场景，显著提升自动驾驶系统的安全性和泛化能力。Waymo表示，该模型将作为其第五代自动驾驶系统的核心组件，未来或向行业合作伙伴开放部分接口。

5. 🚀 X开发者平台上线API按量计费并支持xAI联动

标签： X xAI X开发者平台

X（原Twitter）开发者平台于2026年2月7日宣布推出API按量计费模式，允许开发者根据实际调用量灵活付费，降低初创企业接入门槛。同时，平台深度集成xAI技术栈，支持Grok系列模型调用与实时数据分析功能。此次更新标志着X正从社交平台向AI基础设施服务商转型，进一步强化其在生成式AI应用生态中的战略地位。

6. 🔧 OpenAI开放Sora真人角色视频生成功能

标签： OpenAI Sora

OpenAI于2026年2月7日向Sora用户开放真人角色视频生成功能，支持基于文本提示生成包含真实人物动作与表情的短视频。该功能通过图像到视频（image-to-video）技术实现，允许用户上传参考图像并生成动态演绎内容。尽管仍存在伦理与版权争议，但此举极大拓展了Sora在影视制作、虚拟偶像等领域的应用潜力。

7. 🚀 Perplexity推出多模型聚合问答功能Model Council

标签： Perplexity Model Council

Perplexity于2026年2月7日发布Model Council功能，允许用户同时调用多个大语言模型（如GPT-4、Claude、Llama等）进行协同问答，系统自动整合各模型输出并生成综合答案。该功能旨在提升复杂问题的回答准确性与信息覆盖广度，适用于研究、投资分析等高要求场景。用户可在设置中选择参与聚合的模型组合。

8. 📈 中科曙光启用超三万卡国产AI算力池

标签： 中科曙光 国产AI算力池

中科曙光于2026年2月7日正式启用总规模超过3万张国产AI加速卡的算力池，采用自研DCU架构与液冷散热系统，总算力达120 EFLOPS。该算力池主要服务于国家重大科研项目与重点行业AI训练需求，支持大模型训练、气象模拟、基因分析等高性能计算任务，标志着我国在自主可控AI基础设施领域取得关键突破。

9. 📈 OpenAI内部转向Agent开发模式

标签： OpenAI Agent开发模式

据内部消息披露，OpenAI已于2026年初全面推动研发体系向Agent（智能体）开发模式转型，强调模型在自主规划、工具调用与环境交互方面的能力。此举旨在突破传统LLM的被动响应局限，推动AI系统向主动执行复杂任务演进。相关技术或将应用于即将发布的Dime智能耳机等产品中。

10. 🚀 OpenAI将于2026年推出Dime智能耳机

标签： OpenAI Dime智能耳机

OpenAI计划于2026年内发布首款AI硬件产品Dime智能耳机，集成实时语音翻译、情境感知与个性化助手功能。据多方信源显示，该设备将搭载定制NPU芯片，支持离线运行轻量化模型，并深度整合ChatGPT服务。Dime被视为OpenAI进军消费级AI硬件市场的重要布局，或将重塑智能穿戴设备生态。

11. 🚀 面壁智能发布首款AI硬件松果派

标签： 面壁智能 松果派

面壁智能于2026年2月7日推出首款AI硬件产品“松果派”，定位为面向开发者的边缘计算AI盒子。该设备搭载自研TinyLLM模型，支持本地部署与私有化推理，具备低功耗、高隐私保护特性，适用于智能家居、工业检测等场景。松果派提供SDK与API接口，鼓励社区开发者构建垂直应用。
