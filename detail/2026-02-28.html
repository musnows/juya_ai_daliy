1. 📈 OpenAI完成1100亿美元融资并与亚马逊达成战略合作

标签： OpenAI 亚马逊 Amazon Bedrock

OpenAI宣布完成总额达1100亿美元的新一轮融资，并正式与亚马逊建立战略合作关系。此次合作涵盖技术整合与基础设施支持，OpenAI的先进模型将通过Amazon Bedrock平台提供服务，并引入‘有状态运行时环境’（Stateful Runtime Environment）以增强AI代理的持续交互能力。双方合作还涉及AWS云资源的深度优化，旨在提升大规模AI训练与推理效率。此举标志着OpenAI在保持与微软合作的同时，拓展了关键云合作伙伴生态，有望加速其AI技术在全球企业中的部署。融资资金预计将用于模型研发、算力扩展及安全能力建设。

2. 🔧 Google Antigravity恢复部分账户但重申禁用第三方工具

标签： Google Antigravity Claude Code OpenCode

Google Antigravity项目在短暂中断后恢复部分用户账户访问权限，但官方明确重申禁止使用第三方软件（如Claude Code、OpenClaw、OpenCode等）通过Antigravity登录系统。该政策旨在维护平台安全与数据合规性，防止未经授权的工具接入核心AI开发环境。用户需在官方指定接口内进行操作，违反者可能面临账户封禁。此举反映出谷歌在AI开发工具生态中对安全边界的严格把控。

3. 🔧 Anthropic重置Claude Code用户速率限制

标签： Anthropic Claude Code

Anthropic宣布对Claude Code用户的API调用速率限制进行重置，以优化服务稳定性与资源分配。此次调整可能涉及请求频率、并发连接数或每日配额的重新设定，具体参数未完全公开。此举旨在应对日益增长的开发者使用需求，同时防止系统过载。用户需关注官方文档更新以适配新的调用策略，确保开发流程不受影响。

4. 🚀 谷歌将终止Gemini 3 Pro Preview服务

标签： 谷歌 Gemini 3 Pro Preview Gemini 3.1 Pro Preview

谷歌宣布将于2026年3月9日前逐步终止Gemini 3 Pro Preview服务，并引导用户迁移至Gemini 3.1 Pro Preview版本。新版本在性能、响应速度与多模态理解方面均有提升，支持更复杂的推理任务与长上下文处理。开发者需在截止日期前完成应用迁移，以避免服务中断。此次版本迭代体现了谷歌对Gemini模型系列的持续优化与标准化进程。

5. 📈 Anthropic与美国战争部因AI安全限制爆发冲突

标签： Anthropic 美国战争部

Anthropic就AI模型出口与使用限制问题与美国战争部（Department of War）发生公开分歧。Anthropic发布官方声明，强调其AI系统遵循严格的安全准则，反对将技术用于军事攻击性用途。战争部则质疑部分限制条款妨碍国家安全相关研发。该冲突凸显AI伦理与国防应用之间的紧张关系，可能影响未来AI技术在政府项目中的部署政策。

6. 🚀 OpenAI发布心理健康功能更新并回应诉讼合并

标签： OpenAI

OpenAI推出针对心理健康支持的AI功能更新，增强对话系统对情绪识别与心理危机干预的响应能力。该功能集成于其主流模型中，可提供初步心理支持建议并引导用户寻求专业帮助。同时，OpenAI就多起相关诉讼的合并审理作出正式回应，强调其产品设计符合伦理规范，并将持续优化安全机制。此次更新体现了AI在公共健康领域的应用拓展。

7. 📈 七家科技巨头将签署AI数据中心用电协议

标签： 谷歌 亚马逊 微软

包括谷歌、亚马逊、微软在内的七家科技巨头将在白宫协调下签署一项关于AI数据中心用电的协议，旨在控制电力成本并推动可持续能源使用。该协议由特朗普政府推动，响应AI算力爆发式增长带来的能源压力。企业承诺优化数据中心能效、增加可再生能源采购，并共享最佳实践。此举可能重塑全球AI基础设施的能源战略布局。

8. 🚀 阿里千问AI眼镜将于MWC亮相并开启预约

标签： 阿里巴巴 千问AI眼镜

阿里巴巴旗下千问AI眼镜确认将在世界移动通信大会（MWC）上首次公开展示，并同步开启全球预约通道。该产品集成大语言模型与增强现实（AR）技术，支持实时翻译、信息提示与语音交互功能。设计注重轻便性与续航表现，目标用户涵盖商务人士与科技爱好者。此次发布标志着阿里在消费级AI硬件领域的重要布局。
