3. 🚀 xAI发布Grok 4.1 Fast模型及Agent Tools API

标签： xAI Grok 4.1 Fast Agent Tools API

4. 🚀 Meta发布SAM 3及3D重建模型SAM 3D

标签： Meta SAM 3 SAM 3D

Meta发布新一代图像分割模型SAM 3及该系列首个3D模型SAM 3D，特点是能从单张2D图像中精确重建3D对象和人体。SAM 3在分割精度和速度上均有提升，支持多模态输入。SAM 3D基于SAM架构扩展，实现2D到3D的端到端重建，已开源并可在Hugging Face下载，适用于AR/VR、机器人感知、数字孪生等场景。
