1. 📈 OpenAI紧急开发Garlic模型应对谷歌

标签： OpenAI Garlic GPT

据内部备忘录，OpenAI已进入'红色警报'状态，集中资源改进GPT产品，并推迟广告等非核心项目。公司正在开发代号为Garlic的新模型，计划下周发布新的推理模型。此举旨在应对谷歌在AI领域的快速进展，保持市场竞争力。Garlic项目被视为关键战略举措，以确保技术领先。

2. 🚀 Mistral AI发布Mistral 3模型家族

标签： Mistral AI Mistral 3 Mistral Large 3

Mistral AI发布了新一代开源模型家族Mistral 3，包括专为边缘和本地应用设计的MINI3系列和旗舰模型Mistral Large 3。Mistral Large 3是一个6750亿参数的混合专家模型，具备多模态和多语言能力，适用于复杂推理和生成任务，为企业和开发者提供了强大的开源选择。

🔗 相关链接：
- https://mistral.ai/news/mistral-3

3. 🚀 亚马逊发布Nova 2模型系列及服务

标签： Amazon Nova 2 Nova for

亚马逊扩展其Nova产品组合，发布了四个新的Nova 2模型：轻量级Nova Light、最智能的Nova PRO、语音模型Nova Sonic和统一多模态模型Nova Omni。Nova for支持组织构建自定义模型，Nova act用于创建和部署能在浏览器中执行操作的AI agent。这些服务增强了亚马逊在AI领域的竞争力。

🔗 相关链接：
- https://www.aboutamazon.com/news/aws/aws-agentic-ai-amazon-bedrock-nova-models
- https://nova.amazon.com/dev

4. 📈 Anthropic收购Bun并开源

标签： Anthropic Bun Claude Code

Anthropic宣布收购JavaScript运行时Bun，Bun将成为Claude Code及未来AI编码产品的底层基础设施，并继续保持开源。此次收购旨在提升Anthropic在AI编码领域的能力。同时，Claude Code的年化运营收入已达到10亿美元，显示其市场影响力。

🔗 相关链接：
- https://bun.com/blog/bun-joins-anthropic
- https://www.anthropic.com/news/anthropic-acquires-bun-as-claude-code-reaches-usd1b-milestone

5. 🚀 苹果发布CLaRa-7B-Instruct和STARFlow-V

标签： Apple CLaRa-7B-Instruct STARFlow-V

苹果发布了开源模型CLaRa-7B-Instruct，这是一个基于Mistral-7B微调的统一RG模型，内置语义文档压缩功能。同时，苹果开源了STARFlow-V项目，这是一个基于新型Transformer自回归流架构的文本到视频生成模型，为多模态AI研究提供了新工具。

🔗 相关链接：
- https://huggingface.co/apple/CLaRa-7B-Instruct
- https://github.com/apple/ml-starflow

6. 🔧 谷歌DeepMind发布DiscoRL强化学习算法

标签： Google DeepMind DiscoRL 强化学习

谷歌DeepMind提出名为DiscoRL的新型强化学习算法，通过元学习过程自动发现更新规则，无需人工设计。该算法在复杂任务中表现出色，相关成果已发表在Nature杂志，为强化学习领域提供了新的研究方向。

🔗 相关链接：
- https://www.nature.com/articles/s41586-025-09761-x

7. 🔧 英伟达发布TiDAR AI解码方法

标签： NVIDIA TiDAR AI解码

英伟达发布了名为TiDAR的新型AI解码方法，融合自回归与扩散模型机制，通过单步生成多个token显著提升文本生成速度。该方法在提升生成效率的同时保持高质量输出，为AI文本生成技术提供了新的解决方案。

🔗 相关链接：
- https://arxiv.org/pdf/2511.08923

8. 🚀 商汤开源原生多模态架构NEO

标签： 商汤科技 NEO 多模态架构

商汤科技开源了全新多模态模型架构NEO，这是行业首个实现深层次融合的原生多模态架构。NEO能以极少的数据量达到顶级模型的性能，为多模态AI研究提供了高效解决方案，推动行业技术进步。

🔗 相关链接：
- https://github.com/EvolvingLMMs-Lab/NEO
