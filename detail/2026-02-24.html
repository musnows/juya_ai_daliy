1. 🚀 OpenAI 上线 gpt-realtime-1.5 语音模型

标签： OpenAI gpt-realtime-1.5 Realtime API

OpenAI 在 Realtime API 中正式推出 gpt-realtime-1.5 语音模型，标志着其实时语音交互能力的重要升级。该模型支持低延迟语音对话，具备更强的上下文理解与多轮交互能力，适用于智能客服、实时翻译、语音助手等场景。此次更新强化了 OpenAI 在实时 AI 通信领域的领先地位，并可能推动企业级语音应用的发展。相关技术文档和开发者资源已同步发布。

2. 🔧 Guide Labs 推出可解释因果扩散语言模型 Steerling-8B

标签： Guide Labs Steerling-8B 因果扩散模型

Guide Labs 发布新型可解释因果扩散语言模型 Steerling-8B，强调模型决策过程的可解释性与因果推理能力。该模型通过结构化因果建模提升输出透明度，适用于医疗、金融等高风险决策场景。项目代码已开源，并提供 Hugging Face 模型卡与 TechCrunch 技术解析，推动可解释 AI 研究进展。

3. 🔧 OpenAI Responses API 新增 WebSocket 模式

标签： OpenAI Responses API WebSocket

OpenAI 在其 Responses API 中新增 WebSocket 模式，支持持久化连接与双向实时通信，显著降低延迟并提升交互效率。该功能适用于需要持续对话状态管理的应用，如在线客服、协作工具与实时游戏 AI。开发者可通过官方文档接入，进一步优化用户体验与系统性能。

4. 📈 Google 回应封禁滥用 Antigravity 的 Gemini 订阅用户

标签： Google Gemini Antigravity

Google 对部分滥用 Antigravity 功能的 Gemini 订阅用户实施封禁措施，回应社区关于功能滥用的投诉。Antigravity 作为 Gemini 的一项实验性功能，允许用户生成非常规内容，但存在被用于生成有害信息的风险。此次封禁体现 Google 在 AI 安全与合规方面的持续监管行动。

5. 📈 Anthropic 推广 COBOL 代码自动化迁移方案

标签： Anthropic Claude COBOL

Anthropic 推出基于 Claude 的 COBOL 代码自动化迁移解决方案，帮助企业将老旧 COBOL 系统迁移至现代编程语言。该方案利用 AI 理解复杂业务逻辑并生成可维护代码，降低传统系统现代化成本。此举瞄准金融、政府等依赖遗留系统的行业，推动企业数字化转型。

6. 🚀 Gemini 上线视频模板功能

标签： Google Gemini 视频模板

Google Gemini 新增视频模板功能，用户可通过预设模板快速生成结构化视频内容，适用于营销、教育与社交媒体场景。该功能集成于 Gemini App，支持文本到视频的自动化转换，提升内容创作效率。9to5Google 报道指出，此举强化了 Gemini 在生成式多媒体领域的竞争力。

7. 🔧 openclaw 发布 2026.2.22 版本

标签： openclaw 开源工具

开源项目 openclaw 发布 2026.2.22 版本，更新内容包括性能优化、安全补丁与新功能扩展。该项目专注于 AI 模型部署与管理工具链，此次版本迭代提升了多模型调度效率与系统稳定性，适用于企业级 AI 基础设施。

8. 📈 Anthropic 指控三家中国 AI 实验室进行工业级模型蒸馏攻击

标签： Anthropic 模型蒸馏攻击 Claude

Anthropic 发布报告，指控三家中国 AI 实验室实施大规模模型蒸馏攻击，试图通过 API 调用逆向工程其 Claude 模型。报告指出攻击具备工业级规模，可能威胁模型知识产权与安全。Anthropic 已加强防御机制并呼吁行业关注模型安全。

9. 📈 五角大楼考虑终止与 Anthropic 的 AI 军事合同

标签： 五角大楼 Anthropic AI 军事合同

据 Axios 报道，美国五角大楼正考虑终止与 Anthropic 的 AI 军事合作项目，起因于政策分歧与安全审查。该项目原计划利用 Claude 模型提升军事决策支持能力，但近期因伦理与合规争议面临调整。事件反映 AI 在国防应用中面临的监管挑战。

10. 📈 OpenAI 联手四大咨询巨头组建 Frontier 企业 AI 联盟

标签： OpenAI Frontier 联盟 麦肯锡

OpenAI 宣布与麦肯锡、波士顿咨询、贝恩、德勤四大咨询巨头联合成立 Frontier 企业 AI 联盟，旨在推动企业级 AI 解决方案的规模化落地。联盟将整合 OpenAI 技术与咨询行业经验，加速 AI 在传统行业的渗透与应用。

11. 📈 月之暗面 Kimi 近20天收入超2025全年

标签： 月之暗面 Kimi 大模型商业化

据《澎湃新闻》报道，月之暗面旗下大模型 Kimi 在近20天内实现收入超过2025年全年总额，显示其商业化进程显著加速。增长主要得益于企业 API 调用与订阅服务扩张，反映市场对国产大模型需求的强劲增长。
