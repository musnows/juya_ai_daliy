1. 🚀 Mistral AI发布Devstral 2与Vibe CLI

标签： Mistral AI Devstral 2 Vibe CLI

Mistral AI发布了编程模型Devstral 2及其命令行工具Vibe CLI，包含123B和24B两种参数规模，均为开源。在SWE Bench Verified基准测试中，123B模型得分72.2%，24B模型得分68.0%。Vibe CLI是终端代码自动化工具，支持Devstral 2 API，目前免费开放。12月内，Mistral AI将免费提供这两个模型，以推动开发者生态建设。该工具链适用于自动化代码生成、修复与优化，提升开发效率。

2. 🚀 智谱发布GLM-ASR-Nano-2512语音识别模型

标签： 智谱AI GLM-ASR-Nano-2512 Whisper v3

智谱AI开源了语音识别模型GLM-ASR-Nano-2512，拥有15亿参数，在多个基准测试中性能优于OpenAI Whisper v3。该模型针对粤语等方言和低音量语音进行了优化，提升了在复杂声学环境下的识别准确率。模型支持多语种语音转写，适用于智能客服、会议记录、语音助手等场景。作为轻量级模型，其部署成本低，适合移动端和边缘设备应用。

3. 🚀 智谱开源AutoGLM手机智能助理框架

标签： 智谱AI AutoGLM 风9B

智谱AI开源了手机智能助理框架AutoGLM，具备多模态屏幕理解与自动化操作能力。开源内容包括新模型AutoGLM、风9B完整工具链及50余款中文应用DEMO。该框架支持通过视觉与文本理解用户意图，并自动执行应用操作，如信息查询、表单填写等。目前仅限研究与教育用途，旨在推动AI Agent在移动端的发展。

4. 📈 Linux基金会成立Agentic AI基金会

标签： Linux基金会 Agentic AI基金会 OpenAI

Linux基金会宣布成立Agentic AI基金会，由Anthropic、Block和OpenAI联合创立，旨在推动自主AI（Agentic AI）的透明协作与标准化发展。创始项目包括Anthropic的Claude模型、Block的Bitkey和OpenAI的Agent SDK。该基金会将制定开发规范、安全标准与伦理框架，促进跨组织合作，推动AI Agent生态的健康发展。

5. 🚀 Gemini CLI候补名单本周五关闭

标签： Google Gemini CLI Gemini 3

Google宣布Gemini CLI候补名单将于本周五（12月13日）终止，所有已提交申请的用户将获得Gemini 3的访问权限。免费层用户可在通道关闭前申请，获取Gemini 3的完整功能体验。此举被视为Google为扩大Gemini生态用户基础的重要举措，也是官方免费资格的“最后上车机会”。

6. 🔧 Google Stitch推出Predictive Heatmaps功能

标签： Google Stitch Predictive Heatmaps

Google在其设计工具Stitch中推出Predictive Heatmaps（预测热力图）功能，帮助设计师通过模拟用户视觉行为生成界面关注点热力图。该功能利用AI模型预测用户在界面上的注意力分布，支持设计审计与优化，提升用户体验。适用于网页、APP等界面设计，可减少A/B测试成本，提高设计决策效率。

7. 🚀 Augment Code发布Context Engine SDK

标签： Augment Code Context Engine SDK AI Agent

Augment Code发布Context Engine SDK，允许开发者构建能从代码库、文档、API说明等多源信息中检索上下文的AI Agent与工具。该SDK支持语义检索、代码理解与上下文拼接，适用于智能代码补全、错误诊断、知识问答等场景。开发者可集成至IDE或构建独立工具，提升开发效率与代码质量。

8. 🔧 Qwen Code发布新版本更新说明

标签： Qwen Code Stream JSON 国际化

Qwen Code发布近期版本更新，新增Stream JSON支持，提升流式响应能力；全面支持国际化功能，适配多语言开发环境；修复多项安全与稳定性问题，包括内存泄漏和并发处理缺陷。更新后模型响应更稳定，适用于高并发生产环境。此次更新强化了其在企业级开发工具链中的可用性。

9. 🚀 DiffSynth-Studio发布Qwen-Image-i2L模型

标签： DiffSynth-Studio Qwen-Image-i2L LaTeX

DiffSynth-Studio开源Qwen-Image-i2L模型套件，能将单张或多张图像转换为对应的LaTeX代码，支持风格迁移与快速训练。模型适用于学术图表、公式、表格的自动化生成，支持从图像中提取结构化信息。该套件降低了LaTeX编写门槛，可用于教育、出版、科研等领域，提升内容生产效率。

10. 🚀 Cognitive-Lab发布NetraEmbed多模态嵌入模型

标签： Cognitive-Lab NetraEmbed Co-NetraEmbed

Cognitive-Lab发布多语言多模态嵌入模型NetraEmbed与Co-NetraEmbed，用于解决多语言环境下的多模态文档检索问题。模型在跨语言检索任务上取得先进性能，支持文本、图像、表格等多种模态的联合嵌入，适用于全球化企业文档管理、多语言搜索系统等场景。

11. 🚀 NousResearch开源Nomos-1数学求解模型

标签： NousResearch Nomos-1 Putnam 2025

NousResearch开源数学求解模型Nomos-1，拥有310亿参数，在Putnam 2025数学竞赛中使用专用推理框架得分87/120。该模型专注于高等数学问题求解，支持符号推理、公式推导与证明生成，适用于教育、科研与AI数学助手开发。其开源推动了数学AI的发展。

12. 🚀 One-to-All Animation发布角色动画模型

标签： One-to-All Animation 角色动画模型 姿态迁移

One-to-All Animation发布统一框架，用于高保真角色动画与图像姿态迁移，提供14B和1.3B两种参数规模的模型。框架支持从单张图像生成流畅动画，适用于游戏、影视、虚拟人等领域。其特点是低延迟、高保真，支持多姿态迁移与风格控制，提升内容创作效率。

13. 🚀 inclusionAI发布TwinFlow生成框架

标签： inclusionAI TwinFlow TwinFlow-Image-V1.0

inclusionAI发布TwinFlow框架，通过自对抗流技术实现大型模型的高质量一步生成，无需辅助网络。已发布TwinFlow-Image-V1.0模型，支持图像生成任务。该框架特点是生成速度快、质量高，避免了传统扩散模型的多步迭代，适用于实时图像生成与内容创作。

14. 🚀 stepfun-ai开源PaCoRe推理框架

标签： stepfun-ai PaCoRe HMMR2025

stepfun-ai开源PaCoRe（Parallel Collaborative Reasoning）推理框架，通过并行协同推理扩展测试时计算，在HMMR2025测试上模型准确率达94.5%。该框架支持多模型协同推理，提升复杂任务处理能力，适用于问答、推理、决策等场景，推动大模型推理效率提升。

15. 🔧 Cerebras发布DeepSeek-V3.2压缩模型

标签： Cerebras DeepSeek-V3.2 RIP

Cerebras发布基于DeepSeek-V3.2的两个压缩模型，通过RIP（Redundancy Inference Pruning）方法分别实现25%和50%的专家减值，显著降低部署成本。压缩后模型保持高性能，适用于边缘设备与资源受限环境，推动大模型在工业场景的落地应用。

16. 🔧 Qwen团队介绍SAPO强化学习方法

标签： Qwen SAPO 强化学习

Qwen团队介绍新型强化学习方法SAPO（Smooth Adaptive Policy Optimization），通过平滑的温度控制门控函数替代硬剪切，解决大语言模型强化学习中的策略优化不稳定问题。该方法提升训练稳定性与收敛速度，适用于复杂任务微调，推动RLHF技术发展。

17. 📈 OpenAI发布首批认证课程

OpenAI推出首批认证课程，包括面向企业和公共部门的AI Foundation课程，以及在Coursera上线的ChatGPT Foundation课程。课程涵盖AI基础、安全使用、Prompt工程等内容，旨在提升组织AI素养。完成课程可获得官方认证，推动AI人才体系建设。

18. 🔧 Google测试Veo 3.1模板功能
