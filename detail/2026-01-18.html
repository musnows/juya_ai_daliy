1. 🚀 Vercel 发布 AI 技能管理工具 add-skill

标签： Vercel add-skill

Vercel 于2026年1月18日正式发布其 AI 技能管理工具 add-skill，旨在帮助开发者更高效地集成和管理 AI 功能模块。该工具支持通过命令行或 API 方式快速添加预训练模型、提示模板和推理逻辑到现有项目中，提升 AI 应用开发效率。add-skill 提供模块化架构，支持与 Vercel 平台原生集成，并可扩展至第三方 AI 服务。其 GitHub 仓库已开源，开发者可通过官方链接获取源码与文档。此举标志着 Vercel 进一步拓展其在 AI 开发工具链中的布局，强化其在边缘计算与无服务器 AI 部署领域的竞争力。

🔗 相关链接：
- https://github.com/vercel-labs/add-skill
- https://x.com/rauchg/status/2012345679721771474

2. 🚀 ai-forever 发布 VIBE 图像编辑模型

标签： ai-forever VIBE

俄罗斯 AI 研究机构 ai-forever 于2026年1月18日发布图像编辑模型 VIBE，支持基于文本指令对图像进行局部修改与风格迁移。该模型在 Hugging Face 平台上线，并提供在线演示空间供用户体验。VIBE 采用多模态对齐架构，能够在保持图像语义一致性的前提下实现高精度编辑，适用于内容创作、广告设计等场景。其技术细节通过项目主页公开，展示了在复杂背景与多对象场景下的编辑能力。

🔗 相关链接：

- https://x.com/rauchg/status/2012345679721771474
- https://huggingface.co/iitolstykh/VIBE-Image-Edit

3. 🔧 Cursor 发布新版 CLI 支持 Agent 模式与云端接力

标签： Cursor CLI

代码编辑器 Cursor 于2026年1月18日发布新版命令行工具（CLI），新增 Agent 模式与云端接力功能。Agent 模式允许 CLI 在本地与云端之间自主调度任务，实现代码生成、调试与部署的自动化流程。云端接力功能支持跨设备任务延续，用户可在不同终端无缝切换开发环境。此次更新显著提升了远程协作与持续集成场景下的开发体验，进一步强化 Cursor 作为 AI 编程助手的定位。

🔗 相关链接：
- https://cursor.com/changelog

4. 🔧 Ollama 支持 Anthropic API 及 TranslateGemma 模型

标签： Ollama Anthropic TranslateGemma

开源大模型运行平台 Ollama 于2026年1月18日宣布支持 Anthropic 的 Claude API 接口，并新增 TranslateGemma 模型。用户可通过 Ollama 本地调用 Claude 系列模型，实现低延迟、高隐私的推理服务。TranslateGemma 是基于 Google Gemma 架构优化的翻译专用模型，支持多语言互译，适用于本地化与跨语言内容生成。此次更新扩展了 Ollama 的模型生态，增强其在企业级 AI 部署中的适用性。

🔗 相关链接：
- https://ollama.com/blog/claude
- https://ollama.com/library/translategemma
- https://support.claude.com/en/articles/12138966-release-notes

5. 🔧 Anthropic 为 Claude Team 普通席位开放 Claude Code

标签： Anthropic Claude Code

Anthropic 于2026年1月18日宣布向 Claude Team 计划的普通席位用户开放 Claude Code 功能。该功能此前仅限高级席位使用，现扩展至更广泛用户群体。Claude Code 提供代码补全、错误检测与重构建议，支持主流编程语言与 IDE 集成。此次开放有助于提升团队协作效率，推动 AI 编程工具在企业中的普及。

6. 📈 Anthropic 正为 Claude 开发多项新功能

标签： Anthropic Claude

据社交媒体信息显示，Anthropic 正在为 Claude 模型开发多项新功能，包括增强的上下文理解、多轮对话记忆优化与跨文档推理能力。这些功能预计将提升 Claude 在复杂任务处理与长文本分析中的表现。尽管具体上线时间未公布，但开发进展表明 Anthropic 正持续强化其模型在专业场景中的应用能力。

7. 📈 xAI 全球首个吉瓦级训练集群 Colossus 2 投入运行

标签： xAI Colossus 2

xAI 于2026年1月18日宣布其全球首个吉瓦级（GW-scale）AI 训练集群 Colossus 2 正式投入运行。该集群由超过10万台 GPU 组成，总功耗接近1吉瓦，专为大规模语言模型训练优化。Colossus 2 的启用标志着 xAI 在算力基础设施上的重大突破，有望加速 Grok 系列模型的迭代与性能提升。马斯克在 X 平台确认该集群已全面启用，具备处理万亿参数模型训练任务的能力。
