1. 🚀 Claude 正式上线 100 万 token 上下文，取消阶梯计费

标签： Claude Anthropic

Anthropic 正式推出 Claude 模型的 100 万 token 上下文窗口功能，允许用户处理超长文本输入，如整本电子书或大型代码库，极大提升了模型在长文档理解、分析和生成任务中的能力。同时，Claude 取消了原有的阶梯式定价策略，转向更统一的计费模式，降低了高用量用户的使用门槛。这一更新标志着大模型在处理复杂、长程任务方面的重要突破，尤其利好法律、科研、金融等需要处理海量文本的行业。此次升级可能进一步巩固 Claude 在企业级 AI 应用中的竞争力。

2. 🚀 上海人工智能实验室发布 InternVL-U 统一多模态模型

标签： InternVL-U 上海人工智能实验室

上海人工智能实验室正式推出 InternVL-U 统一多模态模型，该模型支持图像、文本、音频等多种模态的联合理解与生成，具备跨模态对齐与推理能力。InternVL-U 在视觉-语言任务中表现优异，可应用于智能问答、内容审核、多模态搜索等场景，标志着国产多模态大模型技术迈向新阶段。

3. 📈 智谱调整 GLM Coding Plan 折扣，拼好模门槛提高

标签： GLM Coding Plan 智谱 AI

智谱 AI 宣布调整 GLM Coding Plan 的折扣政策，提高了“拼好模”活动的参与门槛，可能涉及更高的使用量要求或更严格的资格审核。此举或旨在优化资源分配，引导用户向更高阶服务迁移，反映出 AI 模型商业化进程中定价策略的动态调整。

4. 🔧 Codex 应用正式上线自动化功能与主题个性化

标签： Codex

Codex 应用正式推出自动化工作流功能，支持用户自定义代码生成与执行流程，同时新增主题个性化设置，提升开发者使用体验。该更新增强了 Codex 在编程辅助中的灵活性与用户粘性，适用于自动化脚本生成、IDE 插件集成等场景。

5. 🔧 Chrome 146 正式发布，支持 MCP 暴露会话供 CLI 使用

标签： Chrome 146 MCP

Google 发布 Chrome 146 版本，正式支持 Model Context Protocol（MCP）功能，允许通过命令行接口（CLI）暴露浏览器会话上下文，为 AI 代理与浏览器自动化交互提供底层支持。这一更新为构建更智能的浏览器自动化工具（如 AI 驱动的网页操作代理）奠定基础。

6. 🚀 CopilotKit 开源 OpenGenerativeUI，支持 AI Agent 生成交互 UI

标签： CopilotKit OpenGenerativeUI

CopilotKit 开源 OpenGenerativeUI 框架，允许 AI Agent 根据自然语言指令动态生成交互式用户界面。该框架支持实时 UI 构建与用户反馈闭环，适用于低代码平台、智能客服系统等场景，推动 AI 驱动的前端开发范式演进。

7. 🚀 Vue 作者尤雨溪推出 Vite 原生部署平台 Void

标签： Void Vite 尤雨溪
