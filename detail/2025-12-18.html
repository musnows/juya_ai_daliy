1. 🚀 Google发布Gemini 3 Flash模型

标签： Google Gemini 3 Flash

Google正式推出Gemini 3 Flash模型，主打极速推理，在多项基准测试中超越了Gemini 2.5 Pro。该模型在编码AIME测试中得分78%，优于Gemini 3 Pro，具备低延迟特性。Gemini 3 Flash已成为搜索AI模式和Gemini应用的默认模型，并向开发者开放API，提供与前代Flash模型一致的免费额度。功能包括自然语言数据处理、网页交互扩展和MCP资源搜索。

🔗 相关链接：
- https://blog.google/products/search/google-ai-mode-update-gemini-3-flash/
- https://blog.google/products/gemini/gemini-3-flash-gemini-app/
- https://blog.google/products/gemini/gemini-3-flash/

2. 🔧 Gemini CLI集成Gemini 3 Flash

标签： Google Gemini CLI Gemini 3 Flash

Google更新Gemini CLI，正式集成Gemini 3 Flash和Pro模型。用户需升级至0.2.1.1版本并开启预览功能。新版CLI增加了使用情况steps命令、自然语言数据分析和网页交互扩展，支持通过命令直接搜索MCP资源，提升了开发者体验和工具链整合能力。

🔗 相关链接：
- https://blog.google/products/search/google-ai-mode-update-gemini-3-flash/
- https://blog.google/products/gemini/gemini-3-flash-gemini-app/
- https://blog.google/products/gemini/gemini-3-flash/

3. 🚀 OpenAI开放ChatGPT应用提交

标签： OpenAI ChatGPT

OpenAI宣布开放ChatGPT应用提交，开发者可上传应用供审核发布，用户可通过新增的应用目录浏览。现阶段开发者可通过链接跳转外部网站完成商品交易，未来将探索更多变现模式。此举标志着ChatGPT生态向第三方应用扩展，推动AI应用商业化进程。

🔗 相关链接：
- https://openai.com/index/developers-can-now-submit-apps-to-chatgpt/

4. 🚀 微软发布TRELLIS.2 3D生成模型

标签： Microsoft TRELLIS.2

微软发布TRELLIS.2 3D生成模型，拥有40亿参数，专为高保真图像转3D设计。该模型采用新型稀疏体素结构，支持生成复杂几何和物理材质，输出高分辨率3D资产。项目代码以MIT协议开源，未来将发布更多预训练权重，适用于游戏、影视和工业设计领域。

🔗 相关链接：
- https://github.com/microsoft/TRELLIS.2
- https://huggingface.co/microsoft/TRELLIS.2-4B
- https://microsoft.github.io/TRELLIS.2/

5. 🚀 腾讯开源HY World 1.5世界模型

标签： 腾讯 HY World 1.5

腾讯混元团队开源HY World 1.5实时世界模型框架，核心WorldPlay模型支持通过文本或图像生成可交互的3D世界，能以每秒24帧的速度实时生成与探索。官方称这是业界最系统的实时世界模型框架，提供在线体验，适用于虚拟场景构建和交互式内容创作。

6. 🚀 通义发布QwenLong-L1.5模型

标签： 通义实验室 QwenLong-L1.5

通义实验室发布QwenLong-L1.5模型，基于Qwen3优化，支持256K物理上下文。通过新型后训练方案和强化学习策略，该模型在长文档推理上提升显著，性能比肩GPT-5和Gemini 2.5 Pro。目前已在Hugging Face上线，适用于复杂文本分析和长上下文任务。

🔗 相关链接：
- https://huggingface.co/Tongyi-Zhiwen/QwenLong-L1.5-30B-A3B

7. 🚀 Browser Use发布BU-30B-A3B模型

标签： Browser Use BU-30B-A3B-Preview

Browser Use发布首个专为浏览器操控设计的开源模型BU-30B-A3B-Preview。该模型基于Qwen3 VR定制，优化了对网页DOM结构的理解和视觉定位能力，支持AI像人类一样浏览网页、点击按钮和提取数据，适用于自动化网页交互和数据采集任务。

🔗 相关链接：

8. 🔧 阶跃星辰升级Step-GUI模型

阶跃星辰升级云端Step-GUI模型，限时免费开放API。新模型支持超200个任务场景，具备更强的语义理解能力。官方开源端侧模型Step-GUI-H，并发布GUIMCP协议，声称可大幅缩短AI手机的部署时间，适用于多模态交互和智能终端应用。

9. 🚀 xAI发布Grok Voice Agent API

标签： xAI Grok Voice Agent API

11. 🔧 蚂蚁开源A-Environment平台

标签： 蚂蚁集团 A-Environment

蚂蚁集团开源A-Environment平台，提出万物皆环境的理念。平台扩展MCP协议，兼容OpenAI Agent SDK，提供开箱即用的环境基础设施和统一API，支持多Agent编排和快速部署，适用于AI应用开发和生产环境构建。

12. 🔧 SGL团队发布迷你SGL推理框架

标签： SGL团队 迷你SGL

SGL团队发布迷你SGL推理框架，代码精简至5000行，保留FlashInfer等核心优化。旨在帮助开发者和学生快速理解大模型推理原理，同时提供与完整版一致的在线服务性能，适用于教学和大模型推理研究。

13. 🔧 昂斯老推出手机端微调方案

标签： 昂斯老 Poch团队

昂斯老联合Poch团队推出手机端大模型微调部署方案，通过量化感知训练，支持将Qwen3等模型部署到iOS或安卓设备，保持较高精度实现本地推理。官方提供完整教程，适用于隐私敏感场景和离线AI应用。

14. 📈 论文揭示模型隐藏意图能力

标签： 神经变色龙

19. 📈 英伟达拟削减RTX50供应
