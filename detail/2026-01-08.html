1. 🚀 OpenAI 发布 ChatGPT Health 独立隐私健康空间

标签： OpenAI ChatGPT Health

OpenAI 正式推出 ChatGPT Health，这是一个专为医疗健康场景设计的独立隐私空间。该功能支持用户连接电子医疗记录和第三方健康应用，基于真实健康数据提供个性化健康建议。开发过程中有260多位职业医生参与，确保医学专业性。数据采用加密隔离技术，不会用于基础模型训练，保障用户隐私安全。目前该功能已向部分用户开放等候名单，标志着AI在个性化医疗助手领域迈出关键一步。

🔗 相关链接：
- https://openai.com/zh-Hans-CN/index/introducing-chatgpt-health/

2. 🔧 Gemini CLI 更新预览版支持 Agent Skills 功能

3. 🚀 OpenBMB 发布 ChatDev 2.0 多智能体协作平台

标签： OpenBMB ChatDev 2.0

OpenBMB 推出 ChatDev 2.0 多智能体平台，并配套上线零代码可视化工具 Devil。用户可通过拖拽方式配置智能体和工作流，处理数据可视化、3D生成等复杂任务。系统采用控制与数据解耦架构，提升灵活性与可维护性，同时提供 Python SDK 供开发者深度集成。该平台显著降低多智能体系统搭建门槛，推动企业级AI应用落地。

4. 🚀 Liquid AI 与 AMD 合作推出本地会议摘要模型 LFM-2

标签： Liquid AI AMD LFM-2

Liquid AI 与 AMD 联合发布专为本地部署设计的会议摘要模型 LFM-2，参数规模26亿。该模型针对 AMD Ryzen AI PC 平台优化，同时支持其他主流推理框架运行，目前仅支持英文转录文本处理。模型强调本地运行、快速响应与完全隐私保护，适用于企业对会议内容进行高效、安全的自动化总结。

5. 🔧 Andrej Karpathy 发布 nanochat 预训练研究成果

标签： Andrej Karpathy nanochat

知名AI研究员 Andrej Karpathy 发布 nanochat 项目预训练阶段成果，通过缩放定律实验验证模型深度对性能的影响。研究表明仅调整模型深度即可实现性能提升，且训练成本极低，部分实验复现仅需约100美元。该项目聚焦计算最优的小型模型训练路径，为资源受限场景下的模型开发提供重要参考。

6. 🚀 字节跳动提出 NextFlow 多模态统一生成模型

标签： 字节跳动 NextFlow

字节跳动发布多模态统一模型 NextFlow，采用分层预测范式，可在5秒内生成高清图像，效率显著优于同类模型。该模型实现图像理解、生成、编辑及视频创作的深度统一，支持端到端多任务处理。其架构设计兼顾性能与通用性，有望在内容创作、设计辅助等领域广泛应用。

🔗 相关链接：
- https://github.com/ByteVisionLab/NextFlow
- https://arxiv.org/abs/2601.02204

7. 📈 中国要求科技公司减少采购英伟达 H200 芯片
