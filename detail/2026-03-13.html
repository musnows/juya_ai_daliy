1. 🚀 GitHub 调整 Copilot 学生计划，取消 GPT-5.4 等模型自选权限

标签： GitHub Copilot GPT-5.4

GitHub 于2026年3月13日宣布对 Copilot 学生计划进行重大调整，取消学生用户自主选择 GPT-5.4 等高级模型的权利。此举意味着学生用户将不再能自由切换底层 AI 模型，可能统一使用默认或指定版本。该调整或出于成本控制与资源管理考量，可能影响学生开发者对模型性能与响应风格的个性化需求。尽管 Copilot 仍提供免费使用，但功能灵活性下降，引发教育用户群体关注。

2. 🔧 Claude 支持生成式 UI，聊天界面构建交互式图表

标签： Claude Anthropic 生成式 UI

Anthropic 公司旗下 Claude 模型正式上线生成式用户界面（Generative UI）功能，用户可通过自然语言指令在聊天界面中直接生成交互式图表。该功能支持动态数据可视化，用户无需导出数据至外部工具即可完成图表创建与交互操作。此举显著提升 Claude 在数据分析与展示场景中的实用性，尤其适用于教育、研究与商业报告场景。

3. 🔧 OpenAI 视频 API 上线新功能

标签： OpenAI 视频 API

OpenAI 正式推出其视频 API 的新功能，具体细节虽未完全披露，但结合上下文推测，可能涉及视频内容生成、编辑或分析能力的增强。该 API 的升级标志着 OpenAI 在多模态 AI 服务领域的进一步扩展，有望为开发者提供更强大的视频处理能力支持。

4. 🚀 NVIDIA 发布 NVILA-8B-HD-Video 模型，支持 4K 视频输入

标签： NVIDIA NVILA-8B-HD-Video

NVIDIA 发布新一代视觉语言模型 NVILA-8B-HD-Video，支持高达 4K 分辨率的视频输入，显著提升对高清视频内容的理解与分析能力。该模型参数规模为80亿，专为高质量视频处理优化，适用于安防监控、影视制作、自动驾驶等对画质要求较高的场景。

5. 🔧 Claude Code 推出语音模式，引入设置脚本功能

标签： Claude Code Anthropic

Claude Code 新增语音交互模式，用户可通过语音指令进行代码编写与调试，提升开发效率。同时引入设置脚本功能，允许用户自定义工作流与自动化任务。该更新增强了 Claude Code 在编程辅助场景中的交互灵活性与可定制性。

6. 📈 Google 调整 Antigravity 计费，Pro 刷新周期延至 7 天

标签： Google Antigravity

Google 对 Antigravity 服务的计费策略进行调整，将 Pro 用户的刷新周期从更短间隔延长至7天。此举可能影响用户对实时数据获取的需求，或为降低服务器负载与运营成本。用户需重新评估其使用频率与数据时效性要求。

7. 🔧 Google Gemini API 现可设置消费上限

标签： Google Gemini API

Google 宣布 Gemini API 支持用户设置消费上限功能，帮助开发者控制 API 调用成本，防止意外超支。该功能适用于企业用户与预算敏感型项目，提升 API 使用的财务可预测性与安全性。

8. 🔧 OpenRouter 上线 Auto Exacto 降低工具调用错误率

标签： OpenRouter Auto Exacto

OpenRouter 推出 Auto Exacto 功能，通过优化工具调用逻辑与参数匹配机制，显著降低 AI 模型在调用外部工具时的错误率。该功能提升多工具协同工作的稳定性，适用于复杂自动化流程与 Agent 应用。

9. 🚀 Hermes Agent v0.2.0 发布，支持多 IDE 集成

标签： Hermes Agent v0.2.0

Hermes Agent 发布 v0.2.0 版本，新增对多种集成开发环境（IDE）的支持，包括 VS Code、JetBrains 系列等。该版本增强了跨平台开发体验，提升 AI 编程助手在不同开发环境中的兼容性与可用性。

10. 🔧 斯坦福大学开源 OpenJarvis 框架支持本地优先个人 AI

标签： 斯坦福大学 OpenJarvis

斯坦福大学发布开源框架 OpenJarvis，专注于构建本地优先的个人 AI 系统，强调数据隐私与离线运行能力。该框架支持用户在不依赖云端服务的前提下部署个性化 AI 助手，适用于对隐私要求较高的个人用户与研究场景。

11. 🚀 谷歌地图集成 Gemini 推出十多年来最大规模升级

标签： Google Maps Gemini Google

谷歌地图完成十年来最大规模的功能升级，全面集成 Gemini 大模型，提升路线规划、地点推荐与实时信息处理能力。新版本支持更自然的语音交互与上下文理解，显著改善用户体验，标志着地图服务向智能化迈进关键一步。

12. 🔧 xAI 发布 Grok 新视频功能，支持 7 张参考图转视频

标签： xAI Grok

xAI 公司为其 Grok 模型推出全新视频生成功能，用户可上传最多7张参考图像，系统据此生成连贯视频内容。该功能拓展了 Grok 在多媒体创作领域的应用边界，适用于短视频制作、概念演示等场景。

13. 🚀 微软推出 Copilot Health 整合设备与健康记录数据

标签： Microsoft Copilot Health

微软正式发布 Copilot Health，该服务整合用户设备数据（如可穿戴设备）与电子健康记录（EHR），提供个性化健康建议与趋势分析。通过 AI 驱动洞察，帮助用户与医疗专业人员更好地管理健康状况，推动数字健康服务智能化。

14. 📈 OpenClaw 生态密集更新，百度联想阶跃发布新服务

标签： OpenClaw 百度 联想 阶跃星辰

OpenClaw 生态系统迎来密集更新，百度、联想与阶跃星辰相继发布基于 OpenClaw 的新服务。这些更新涵盖模型优化、工具集成与应用场景拓展，反映 OpenClaw 在开源 AI 生态中的快速扩张与产业影响力提升。

15. 🚀 Perplexity 向Pro 用户开放 Perplexity Computer

标签： Perplexity Perplexity Computer

Perplexity 宣布向 Pro 订阅用户开放 Perplexity Computer 功能，允许用户通过自然语言指令控制计算机操作，如文件管理、网页浏览与应用调用。该功能标志着 AI 助手向操作系统级集成的重大突破。

16. 🔧 淘宝 PC 端支持 MCP，可接入 OpenClaw

标签： 淘宝 MCP OpenClaw

淘宝 PC 端正式支持模型调用协议（MCP），用户可通过该协议接入 OpenClaw 生态中的 AI 服务。此举提升淘宝平台与第三方 AI 工具的互操作性，为用户提供更丰富的智能购物与内容生成体验。

17. 🚀 ComfyUI 发布 App Mode 及 ComfyHub 支持工作流分享

标签： ComfyUI ComfyHub

ComfyUI 推出 App Mode 模式，优化用户界面与操作体验，同时上线 ComfyHub 平台，支持用户分享与下载 AI 工作流。该更新降低 AI 内容创作门槛，促进社区协作与知识共享。

18. 🔧 Canva 推出 Magic Layers 支持平面图转多层可编辑

标签： Canva Magic Layers

Canva 发布 Magic Layers 功能，用户可将上传的平面图自动转换为多层可编辑设计稿，支持独立调整各图层元素。该功能极大简化设计流程，提升非专业用户的设计效率与创作自由度。

19. 🔧 Cursor 发布 CursorBench 评测体系，优化 Agent 编程评估

标签： Cursor CursorBench

Cursor 公司推出 CursorBench 评测体系，专为评估 AI Agent 在编程任务中的表现而设计。该体系涵盖代码质量、逻辑正确性与任务完成度等维度，为开发者提供标准化评估工具，推动编程 Agent 的优化与比较。

21. 📈 腾讯回应 OpenClaw 数据争议：称仅拉取 1GB 分担流量
