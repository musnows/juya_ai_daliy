1. 📈 OpenAI 官方宣布关停 Sora 应用

标签： OpenAI Sora

OpenAI 于2026年3月25日正式宣布关闭其AI视频生成应用 Sora。该决定标志着这一曾引发广泛关注的生成式AI视频工具正式退出市场。尽管 Sora 在发布初期展示了强大的文本转视频能力，能够根据用户输入的提示词生成高质量、高保真度的视频内容，但其在实际运营中可能面临技术、成本或战略调整等挑战。此次关停反映出AI视频生成领域的高门槛与激烈竞争，也表明OpenAI正在重新评估其产品线布局。Sora 的关闭可能影响依赖该工具进行内容创作的用户群体，并促使市场转向其他替代方案。

2. 📈 LiteLLM 遭遇供应链攻击，两版本含窃凭证恶意代码

标签： LiteLLM

开源AI工具 LiteLLM 被发现遭遇供应链攻击，其两个版本中被植入窃取凭证的恶意代码。该攻击通过污染软件包依赖链实现，可能导致用户API密钥等敏感信息泄露。事件凸显了AI开发工具生态中供应链安全的重要性，建议用户立即检查所用版本并更新至安全补丁。此次攻击可能影响使用 LiteLLM 进行模型调用的开发者和企业，引发对第三方AI库安全审计的广泛关注。

3. 🚀 阿里通义实验室发布 PrismAudio，论文入选 ICLR 2026

标签： 阿里巴巴 通义实验室 PrismAudio

阿里巴巴通义实验室正式发布音频理解与生成模型 PrismAudio，相关研究论文已被国际顶级机器学习会议 ICLR 2026 接收。该模型在音频语义理解、多模态对齐及语音合成方面表现突出，支持高精度语音识别与情感化语音生成。PrismAudio 的发布进一步丰富了通义系列多模态AI产品线，有望应用于智能客服、内容创作、无障碍交互等场景，推动音频AI技术的产业化落地。

4. 🚀 Ai2 发布 MolmoWeb 开源视觉模型及数据集

标签： Ai2 MolmoWeb

人工智能研究机构 Ai2（Allen Institute for AI）正式发布开源视觉语言模型 MolmoWeb，并同步开放配套训练数据集。该模型专注于网页内容理解，具备强大的图像-文本联合推理能力，可用于网页结构解析、视觉问答、自动化测试等任务。作为开源项目，MolmoWeb 降低了视觉AI技术的应用门槛，促进学术界与工业界在网页智能处理方向的合作与创新。

5. 🔧 英伟达开源 Kimodo 动作生成工具，适配类人机器人

标签： 英伟达 Kimodo

英伟达发布并开源动作生成工具 Kimodo，专为类人机器人设计，支持基于自然语言指令生成复杂、流畅的肢体动作序列。该工具结合强化学习与运动规划技术，显著提升机器人在真实环境中的动作适应性与安全性。Kimodo 的开源将加速人形机器人研发进程，推动其在家庭服务、工业协作等场景的应用落地。

6. 🚀 京东云 JoyBuilder 上线 AI 编程订阅服务

标签： 京东云 JoyBuilder

京东云正式推出 JoyBuilder AI 编程订阅服务，为用户提供基于大模型的代码生成、调试与优化功能。该服务支持多种编程语言与主流开发框架，可集成至现有开发流程中，提升开发效率。订阅模式降低了企业使用AI编程工具的门槛，标志着AI辅助开发正从实验阶段走向规模化商用。

7. 🔧 Anthropic 发布 Claude Code 自动模式，支持安全权限自主决策

标签： Anthropic Claude Code

Anthropic 为其 AI 编程工具 Claude Code 推出“自动模式”，允许模型在预设安全权限范围内自主执行代码修改、测试与部署任务。该模式通过强化权限控制与行为审计机制，确保AI操作的可控性与安全性。此举标志着AI编程助手向更高自主性迈进，有望提升软件开发的自动化水平。

8. 🔧 Adobe 升级 Firefly 平台，推出自定义模型功能

标签： Adobe Firefly

Adobe 对其生成式AI平台 Firefly 进行重大升级，新增自定义模型训练功能，允许企业用户基于自有数据微调专属AI模型。该功能支持图像、视频与文本生成，满足品牌一致性、版权合规等商业需求。升级后的 Firefly 平台进一步巩固了Adobe在创意AI领域的领先地位，助力企业实现个性化内容生产。

9. 🚀 Linear 称 Issue Tracking 已死，推出 Agent 新功能

标签： Linear

项目管理工具 Linear 发布声明称传统 Issue Tracking 模式已过时，并推出基于AI Agent 的新功能，可自动识别任务依赖、分配优先级并生成执行建议。该Agent具备上下文理解与长期记忆能力，能够主动推进项目进展。此举代表项目管理工具向智能化、自动化方向演进，可能重塑团队协作方式。

10. 🔧 Anthropic 介绍 Harness 方案，优化 Claude 长流程开发能力

标签： Anthropic Claude Harness

Anthropic 发布 Harness 开发框架，旨在提升 Claude 模型在长流程、多步骤任务中的稳定性与可控性。该方案通过任务分解、状态管理与错误恢复机制，增强AI在复杂应用开发中的可靠性。Harness 的推出有助于开发者构建更健壮的AI驱动应用，推动Claude在工业级场景中的落地。

11. 📈 国家数据局明确词元命名规范，日均调用量两年增长超千倍

标签： 国家数据局

国家数据局正式发布词元（Token）命名与计量标准，统一AI模型输入输出单位的定义与统计方式。数据显示，自标准实施以来，全国AI系统日均词元调用量在两年内增长超过1000倍，反映AI应用规模的爆发式扩张。该规范有助于提升行业透明度，促进算力资源合理配置与成本核算。

12. 📈 OpenAI 基金会启动运作，计划一年投资十亿美元推进 AGI

13. 📈 OpenAI 调整 ChatGPT 电商战略，弱化结账功能转向产品发现

标签： OpenAI ChatGPT

OpenAI 调整 ChatGPT 在电商领域的战略方向，逐步弱化内置结账功能，转而强化其作为“产品发现引擎”的角色。新版本将更注重商品推荐、比价分析与购物咨询，避免与电商平台直接竞争。这一调整有助于ChatGPT与零售商建立更开放的合作关系，拓展其在消费决策链中的价值。
