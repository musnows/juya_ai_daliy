1. 🚀 Cursor 承认 Composer 2 基于 Kimi K2.5，月之暗面确认授权合作

标签： Cursor Kimi K2.5 月之暗面

Cursor 公司正式承认其 AI 编程工具 Composer 2 基于月之暗面（Moonshot AI）的 Kimi K2.5 模型开发，月之暗面方面也确认双方已达成正式授权合作。此举标志着国内大模型在开发者工具领域的深度应用落地，Kimi K2.5 凭借其强大的代码理解与生成能力，为 Composer 2 提供了核心智能支持。该合作有助于提升 Cursor 在代码补全、重构与上下文感知方面的表现，进一步巩固其在 AI 编程助手市场的竞争力。此次技术整合也体现了大模型厂商与工具平台之间日益紧密的生态协作趋势。

2. 🔧 Claude Code 上线云端定时任务，桌面版支持直接选择 DOM

标签： Claude Code Anthropic

Anthropic 推出的 AI 编程工具 Claude Code 新增云端定时任务功能，用户可设定周期性代码检查或自动化脚本执行。同时，桌面版本更新支持开发者直接在界面中选择网页 DOM 元素，实现更精准的上下文感知与代码生成。这一改进显著提升了前端开发效率，使 Claude Code 在网页交互类任务中具备更强的实用性，进一步缩小与 Cursor、Windsurf 等竞品的功能差距。

3. 📈 智谱与小米确认将继续推进开源模型战略

标签： 智谱 AI 小米

智谱 AI 与小米公司均公开表示将继续坚持开源模型路线，推动大模型技术的普惠化与社区共建。尽管当前行业存在商业化闭源趋势，两家企业强调开源对技术迭代与生态繁荣的重要性。此举有望激励更多开发者参与模型优化与应用创新，尤其在中文 NLP 与端侧部署场景中形成差异化优势。

4. 🚀 美团开源 LongCat-Flash-Prover 数学推理模型

标签： 美团 LongCat-Flash-Prover

美团正式发布并开源其自研的数学定理证明模型 LongCat-Flash-Prover，该模型专注于形式化数学推理与自动证明任务，具备高精度逻辑推导能力。开源后将促进学术界与工业界在自动推理领域的研究进展，尤其对教育科技、科研辅助等场景具有潜在应用价值。

5. 🔧 英伟达发布 30B MoE 模型，IMO 与 IOI 竞赛获金牌成绩

标签： 英伟达 MoE 模型

英伟达推出一款拥有 300 亿参数规模的混合专家（MoE）架构大模型，在数学推理基准测试中表现优异，成功在国际数学奥林匹克（IMO）与国际信息学奥林匹克（IOI）模拟题集中取得金牌级别成绩。该模型展示了 MoE 架构在复杂逻辑与符号推理任务上的强大潜力，为 AI 在高端学术竞赛中的应用提供了新范例。

6. 🚀 Windsurf 向付费用户限免开放 Kimi K2.5 模型

标签： Windsurf Kimi K2.5

AI 编程工具 Windsurf 宣布向所有付费用户免费开放 Kimi K2.5 模型的使用权限，此前该模型仅限特定测试用户使用。此举旨在提升用户体验与产品竞争力，利用 Kimi K2.5 强大的多模态理解与长上下文处理能力，增强代码生成质量与项目级理解能力。

7. 🚀 OpenAI 推出面向学生的 Codex for Students 项目

标签： OpenAI Codex for Students

OpenAI 启动 Codex for Students 专项计划，为在校学生提供免费或低成本的 AI 编程辅助服务，涵盖代码解释、调试建议与学习资源推荐等功能。该项目旨在降低编程学习门槛，推动 AI 技术在教育领域的公平应用，预计将惠及全球数百万学生开发者。

8. 📈 心流团队宣布 iFlow CLI 服务将于 2026 年 4 月关闭

标签： 心流团队 iFlow CLI

心流团队发布公告，其命令行工具 iFlow CLI 将于 2026 年 4 月正式停止服务，用户需在此之前完成数据迁移与替代方案部署。iFlow CLI 曾用于自动化工作流管理与 AI 任务调度，其关停可能影响部分依赖该工具的开发者生态。

9. 🔧 Claude Cowork 推出 Projects 功能支持团队协作

标签： Claude Cowork Anthropic

Anthropic 的协作平台 Claude Cowork 新增 Projects 功能，支持团队成员在统一空间中共享提示词、文档与对话历史，实现更高效的 AI 辅助协作。该功能面向企业用户设计，有助于提升跨部门项目协同效率，是 AI 办公工具向深度集成迈出的重要一步。

10. 🚀 腾讯 QClaw 开启全量公测，免邀请码打通五大平台

标签： 腾讯 QClaw

腾讯 AI 助手 QClaw 正式开启全量公测，取消邀请码限制，并实现与微信、QQ、企业微信、腾讯会议及腾讯文档五大平台的无缝打通。用户可在多端同步使用 AI 功能，包括内容生成、信息提取与智能摘要，标志着腾讯在 AI 助手生态整合方面取得关键进展。

11. 🔧 Lovable 拓展通用任务处理能力

标签： Lovable

AI 应用生成平台 Lovable 更新其核心引擎，增强对通用任务的解析与执行能力，支持更复杂的用户指令与多步骤工作流。此次升级使其不仅能生成简单网页，还可处理数据整理、表单逻辑设计等中级复杂度任务，拓宽了无代码开发的应用边界。

12. 🔧 OpenAI 构建 AI 研究员，2028 年拟发布全自动化系统

标签： OpenAI AI 研究员

OpenAI 正在内部研发具备自主研究能力的 AI 研究员系统，目标是在 2028 年前推出可全自动完成科学发现、论文撰写与实验设计的综合平台。该系统将整合多模态推理、文献分析与假设生成能力，若实现将极大加速科研进程，但也引发关于学术诚信与人力替代的广泛讨论。

13. 📈 蓝色起源申请部署近五万二千颗 AI 计算卫星

标签： 蓝色起源 FCC

杰夫·贝索斯创立的蓝色起源公司向美国联邦通信委员会（FCC）提交申请，计划部署总计 51,840 颗专用 AI 计算卫星，构建天基 AI 算力网络。该星座旨在为全球提供低延迟、高带宽的 AI 推理与训练服务，尤其服务于边缘计算与实时决策场景，是太空算力竞赛的重要一步。

14. 🚀 豆包开启 AI 电商功能内测，预计 3 月正式上线

标签： 豆包 字节跳动

字节跳动旗下 AI 助手豆包启动电商功能内测，用户可通过自然语言指令完成商品搜索、比价、下单与售后咨询等操作。该功能整合多平台电商数据，结合个性化推荐算法，预计将于 2026 年 3 月正式向公众开放，标志着 AI 助手向交易闭环延伸。

15. 📈 元宝宣布品牌标识升级，以全新形象面向用户

标签： 元宝 百度

百度 AI 助手元宝完成品牌视觉系统全面升级，启用全新 Logo 与界面设计，强化其作为个人智能助手的定位。此次品牌焕新伴随功能优化与用户体验提升，旨在增强用户认知与市场辨识度，进一步巩固其在中文 AI 助手市场的领先地位。

16. 🔧 微软优化 Windows 11，削减 Copilot 集成并允许暂停更新

18. 📈 深度求索核心研究员郭达雅离职，或将加盟大厂负责 Code 业务
