1. 🚀 谷歌DeepMind发布实验性AI世界模型Project Genie

标签： 谷歌DeepMind Project Genie

谷歌DeepMind于2026年1月30日正式发布实验性AI世界模型Project Genie。该模型旨在构建能够理解和模拟复杂环境的生成式世界模型，具备从少量示例中学习并生成动态交互场景的能力。Project Genie支持通过文本或图像提示生成可交互的虚拟环境，可用于游戏开发、机器人训练和模拟系统构建。其核心技术基于大规模多模态学习与强化学习框架，强调对物理规则和因果关系的建模。该项目目前处于实验阶段，相关技术文档和演示已在Google Labs及DeepMind官网发布。此举标志着谷歌在通用人工智能（AGI）路径上对‘世界模型’方向的重要探索，有望推动AI在具身智能与仿真环境中的长期发展。

2. 🚀 通义千问开源Qwen3-ASR语音识别模型

标签： 通义千问 Qwen3-ASR

阿里巴巴通义千问团队于2026年1月30日开源Qwen3-ASR模型，该模型专注于高精度自动语音识别（ASR）任务。Qwen3-ASR支持多语言、多方言语音转文本，具备低延迟、高鲁棒性特点，适用于实时语音交互、会议转录、无障碍辅助等场景。模型已在GitHub、Hugging Face和ModelScope平台同步开源，并提供预训练权重与推理接口。其架构融合了Transformer与流式处理技术，在公开语音数据集上表现优于同类开源模型。此次开源进一步推动阿里在AI基础模型生态中的布局，助力开发者构建低成本、高效率的语音应用。

3. 📈 OpenAI取消API验证并澄清用户发现所有权政策

标签： OpenAI

OpenAI于2026年1月30日通过官方开发者账号宣布取消部分API调用验证机制，并澄清其不会主张对用户在使用其平台过程中所发现或创造的内容的所有权。此前因 monetization plans 引发社区对用户成果归属的担忧，OpenAI明确表示用户对其通过API生成的发现、创意或衍生作品保留完整权利。此举旨在增强开发者信任，促进AI应用创新。相关政策调整即时生效，适用于所有OpenAI API用户。

4. 🔧 Google DeepMind发布基因组预测模型AlphaGenome

标签： Google DeepMind AlphaGenome

Google DeepMind在《自然》杂志发表论文，正式发布基因组预测模型AlphaGenome。该模型基于深度学习技术，能够高效预测基因序列功能、变异影响及潜在致病性，显著提升基因组学研究的分析效率。AlphaGenome在多个基准测试中超越现有工具，支持大规模基因组数据处理，有望加速精准医疗、遗传病诊断和生物制药研发进程。研究成果标志着AI在生命科学领域应用的又一重要突破。

5. 🚀 Skywork AI开源SkyReels-V3多模态视频生成模型

标签： Skywork AI SkyReels-V3

Skywork AI于2026年1月30日开源SkyReels-V3多模态视频生成模型。该模型支持文本、图像到视频的端到端生成，具备高保真画质与连贯动作表现，适用于短视频创作、虚拟内容制作等场景。模型已在GitHub和Hugging Face平台发布，并提供在线演示空间。SkyReels-V3采用扩散模型与时空注意力机制，显著提升视频生成的稳定性与可控性，推动开源视频生成技术的发展。

6. 🚀 OpenMOSS开源MOVA同步音视频生成模型

标签： OpenMOSS MOVA

OpenMOSS团队于2026年1月30日开源MOVA同步音视频生成模型。该模型能够实现语音与口型、表情的精准同步，适用于虚拟人、数字主播、在线教育等场景。MOVA支持多语言输入与个性化音色克隆，提供360p分辨率输出版本。模型代码与权重已在GitHub、Hugging Face及MOSI平台同步发布，助力开发者构建高拟真度音视频交互系统。

7. 🔧 飞桨发布PaddleOCR-VL-1.5视觉语言OCR模型

标签： 飞桨 PaddleOCR-VL-1.5

百度飞桨团队于2026年1月30日发布PaddleOCR-VL-1.5版本，该模型融合视觉与语言理解能力，支持复杂版面文档的端到端识别与结构化解析。PaddleOCR-VL-1.5在表格、公式、多栏文本等场景下表现优异，适用于金融、教育、政务等领域的文档数字化处理。模型已在GitHub和Hugging Face开源，并提供轻量化部署方案，推动OCR技术向多模态智能化演进。

8. 🚀 MiniMax发布Music 2.5音乐生成模型

标签： MiniMax Music 2.5

MiniMax于2026年1月30日发布Music 2.5音乐生成模型。该模型支持多风格、多乐器音乐创作，用户可通过文本描述或旋律片段生成高质量音频。Music 2.5具备长时程结构保持能力，适用于背景音乐制作、游戏音效、创意配乐等场景。模型API已集成至MiniMax平台，提供开发者友好的调用接口，推动AI音乐生成技术的商业化应用。

9. 🚀 xAI正式发布Grok Imagine API图像生成接口

标签： xAI Grok Imagine API

xAI于2026年1月30日正式发布Grok Imagine API，提供基于Grok模型的图像生成服务。该API支持文本到图像的高分辨率生成，具备风格控制与语义理解能力，适用于创意设计、内容营销、原型设计等场景。开发者可通过x.ai文档平台接入API，实现快速集成。Grok Imagine的发布标志着xAI在生成式AI服务生态中的进一步扩展。

10. 🔧 Cursor发布AI代码溯源标准Agent Trace

标签： Cursor Agent Trace

Cursor团队于2026年1月30日发布AI代码溯源标准Agent Trace。该标准旨在追踪AI生成代码的来源、修改路径与依赖关系，提升代码透明度与可审计性。Agent Trace提供开源参考实现与规范文档，支持与主流IDE集成，适用于企业级AI编程助手的安全合规需求。此举推动AI辅助开发向可解释、可追溯方向发展。

11. 🔧 Gemini API上线Gemini 3系列模型与Computer Use工具

标签： Google Gemini API Computer Use

Google于2026年1月30日在Gemini API中正式上线Gemini 3系列模型，并集成Computer Use工具。该工具允许AI模型通过API控制计算机界面，执行点击、输入、文件操作等任务，实现端到端自动化。Gemini 3系列在推理能力、多模态理解与长上下文处理方面显著增强，适用于智能助手、自动化办公与复杂工作流场景。相关功能已在ai.google.dev开放预览。

12. 🚀 Gradio团队发布Daggr助力AI工作流开发

标签： Gradio Daggr

Gradio团队于2026年1月30日发布Daggr工具，旨在简化AI应用工作流的构建与部署。Daggr提供可视化编排界面与模块化组件，支持快速集成模型、数据处理与用户交互逻辑，降低AI应用开发门槛。该工具适用于原型设计、内部工具搭建与教育场景，进一步推动Gradio在AI开发者社区中的生态影响力。
