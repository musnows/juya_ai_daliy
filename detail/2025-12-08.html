1. 🚀 Gemini 3 Flash 或已上线 LM Arena

标签： Google DeepMind Gemini 3 Flash LM Arena

Google DeepMind 的 Gemini 3 Flash 模型疑似已上线 LM Arena 平台，代号为 SKY 和 C 号克，引发业界对其即将正式发布的广泛猜测。该模型作为 Gemini 3 系列中的轻量高速版本，预计将具备低延迟、高吞吐特性，适用于实时对话、边缘推理等场景。结合 LM Arena 的匿名测试机制，用户已可体验其响应速度与多轮对话能力。若属实，这标志着 Google 在轻量化大模型部署上迈出关键一步，或将在移动端、自动驾驶等领域实现更深度集成。正式发布预计将在2025年内完成。

🔗 相关链接：
- https://x.com/AILeaksAndNews/status/1997795213650415875

2. 🚀 NewBieAi-lab 发布动漫图像生成模型

标签： NewBieAi-lab NewBie-image-Exp0.1 ModelScope

NewBieAi-lab 发布开源图像生成模型 NewBie-image-Exp0.1，参数规模为3.5B，专为高质量动漫图像生成设计。模型特点是能处理复杂多角色场景，支持精细构图与风格一致性。功能包括角色表情控制、背景细节生成和动态姿势建模。模型权重仅限非商业用途，源代码采用 Apache 2.0 许可，可在 ModelScope 平台获取。该模型为动漫创作者、独立游戏开发者提供低成本、高效率的生成工具，推动二次元内容创作民主化。

🔗 相关链接：
- https://www.modelscope.cn/models/NewBieAi-lab/NewBie-image-Exp0.1

3. 🚀 ServiceNow 发布多模态推理模型 Apriel-1.6-15B-Thinker

标签： ServiceNow Apriel-1.6-15B-Thinker Hugging Face

ServiceNow 发布多模态推理模型 Apriel-1.6-15B-Thinker，参数为15B，特点是性能可媲美10倍参数量（约150B）的模型。该模型支持文本与图像输入，功能包括复杂逻辑推理、跨模态理解与任务规划。关键优势在于推理 token 使用量减少30%以上，且可在单个消费级 GPU 上高效运行，显著降低部署成本。适用于企业自动化、客服系统、知识管理等场景，已在 Hugging Face 开源，推动轻量化推理模型在产业端落地。

🔗 相关链接：
- https://huggingface.co/ServiceNow-AI/Apriel-1.6-15b-Thinker

4. 🚀 LLM360 发布完全开放模型 K2-V2-Instruct

标签： LLM360 K2-V2-Instruct GitHub

LLM360 发布70B参数大模型 K2-V2-Instruct，是其家族中功能最强大的完全开放模型之一。项目特点是公开了所有训练数据、代码、检查点与模型权重，支持端到端复现。模型支持多轮对话、指令遵循、代码生成与逻辑推理，适用于科研、教育及企业级应用。训练过程透明，涵盖从预训练到对齐的完整流程，已在 Hugging Face 和 GitHub 同步开源，推动大模型研究的可复现性与可信度。

🔗 相关链接：
- https://huggingface.co/LLM360/K2-V2-Instruct
- https://github.com/llm360/k2v2_train

5. 🔧 ImageCritic 发布图像后编辑框架

标签： ImageCritic Hugging Face CCBY-NC 4.0

ImageCritic 是一个参考引导的图像后编辑框架，用于修复 AI 生成图像中的乱码、文本扭曲、Logo 错位等细节问题。框架特点是基于参考图像进行局部优化，支持像素级修复与语义一致性保持。功能包括文本校正、边缘平滑、对象对齐与风格迁移。采用 CCBY-NC 4.0 许可，仅限非商业用途。已在 Hugging Face 和项目主页发布，适用于图像生成后处理、设计辅助与内容审核，提升生成图像可用性。

🔗 相关链接：
- https://huggingface.co/ziheng1234/ImageCritic
- https://ouyangziheng.github.io/ImageCritic-Page/

6. 🔧 Grok 深度集成 Tesla 并预告 4.20 版本

标签： xAI Grok Tesla

xAI 的 Grok 模型已深度集成至 Tesla 车辆系统，新增功能包括作为智能驾驶助手接受导航指令、语音交互与实时路况分析。Grok 4.20 版本预计将在3~4周后发布，将进一步提升多模态理解与上下文记忆能力。集成后，车主可通过自然语言控制车辆功能，如路线规划、充电站推荐与驾驶模式切换。该集成标志着 AI 助手从信息交互向物理世界控制的关键跨越，推动智能座舱发展。

7. 🚀 Stack Overflow 发布 AI Assist 工具

标签： Stack Overflow AI Assist 生成式 AI

Stack Overflow 发布 AI Assist 工具，结合生成式 AI 与社区17年积累的专家知识库，提供经过人类验证的编程答案。工具特点是答案来源透明，标注社区贡献者，支持代码片段生成、错误调试与最佳实践推荐。功能包括上下文感知问答、多语言支持与 API 集成。已在 stackoverflow.com/ai-assist 上线，面向现代开发者提供高效、可信的技术支持，缓解 AI 幻觉问题，提升开发效率。

🔗 相关链接：

- https://stackoverflow.com/ai-assist

8. 📈 Google 扩大 TPU 供应与数据中心部署

标签： Google TPU FlexStak

Google 正扩大 TPU 供应，与 FlexStak 合作在纽约数据中心托管其 TPU 集群。计划到2027年生产超过500万颗 TPU，支持 AI 训练与推理需求。TPU 部署将覆盖全球多个数据中心，提升 Gemini 等模型训练效率。Google 还通过 The Information 披露，TPU 项目估值达20亿美元，总融资4.5亿美元，资金将用于产品路线图推进与全球基础设施扩张，强化其在 AI 芯片领域的领先地位。

9. 📈 d-Matrix 完成 2.75 亿美元 C 轮融资

标签： d-Matrix Nighthawk AI 推理芯片

AI 芯片公司 d-Matrix 完成 C 轮融资，金额达2.75亿美元，总融资额达4.5亿美元，估值20亿美元。资金将用于推进其 AI 推理芯片产品路线图，包括下一代 Nighthawk 芯片开发与全球市场扩张。d-Matrix 专注于低功耗、高吞吐的推理芯片，适用于边缘计算与数据中心。其技术特点是采用数字内存计算（DMC）架构，显著降低推理延迟与能耗，推动 AI 推理从云端向边缘迁移。
