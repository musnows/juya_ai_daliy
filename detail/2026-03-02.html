1. 🚀 Cognition 发布 SWE-1.6 模型早期预览版

标签： Cognition SWE-1.6

AI 公司 Cognition 于2026年3月2日发布了其 SWE-1.6 模型的早期预览版本。该模型专注于软件工程任务，旨在提升代码理解、生成与修复能力。此次预览版面向开发者开放，允许用户通过官方博客链接（https://cognition.ai/blog/swe-1-6-preview）体验其核心功能。SWE-1.6 在代码补全、错误检测与自动化重构方面表现突出，支持多语言编程环境，并采用强化学习优化代码生成质量。该模型的推出标志着 AI 在专业软件开发领域的进一步深入，有望降低开发门槛、提升工程效率。Cognition 表示，正式版将集成更多企业级安全与协作功能。

2. 🚀 Claude 登顶 App Store 并上线记忆导入功能

标签： Anthropic Claude

Anthropic 开发的 AI 助手 Claude 在2026年3月1日跃升至 App Store 第二名，成为当日下载量最高的 AI 应用。与此同时，Claude 正式推出“记忆导入”功能（https://claude.com/import-memory），允许用户将过往对话、偏好设置与个性化数据迁移至新设备或账户，提升用户体验连续性。该功能支持加密传输与本地存储选项，符合隐私保护标准。此次排名上升被认为与其在五角大楼争议事件后的公众关注度提升有关，TechCrunch 报道指出，尽管面临政策质疑，Claude 的用户增长势头强劲。

3. 🔧 Anthropic 研究揭示 LLM 低成本去匿名化风险

标签： Anthropic LLM

Anthropic 发布最新研究论文（arXiv:2602.16800），指出大型语言模型（LLM）存在被用于低成本去匿名化的安全威胁。研究表明，攻击者可通过分析模型输出中的语言风格、用词习惯等细微特征，以极低成本识别匿名文本背后的真实用户身份。该研究基于对多个开源与闭源模型的测试，发现即使经过数据脱敏处理，部分模型仍保留可追踪的个体语言指纹。这一发现对隐私保护政策与模型部署安全提出新挑战，Anthropic 呼吁行业加强输出过滤与差分隐私机制。

4. 🔧 Anthropic 分享 Claude Code 工具开发经验

标签： Anthropic Claude Code

Anthropic 工程师在社交媒体平台 X 上分享了构建 Claude Code 工具的技术经验（https://x.com/trq212/status/2027463795355095314）。Claude Code 是一款面向开发者的 AI 编程助手，集成于 IDE 环境中，支持实时代码建议、上下文理解与自动化测试生成。团队强调在模型微调、上下文窗口优化与用户反馈闭环方面的关键实践，特别是在处理长代码文件时的记忆管理机制。此次经验分享为 AI 编程工具的开发提供了实用参考，推动行业技术透明度提升。

5. 📈 美军在空袭伊朗行动中使用 Claude 模型

标签： Anthropic Claude 美军

据《华尔街日报》实时报道（https://www.wsj.com/livecoverage/iran-strikes-2026/card/u-s-strikes-in-middle-east-use-anthropic-hours-after-trump-ban-ozNO0iClZpfpL7K7ElJ2），美军在2026年对伊朗的空袭行动中使用了 Anthropic 的 Claude 模型进行情报分析与决策支持。该部署发生在特朗普政府短暂禁止军方使用商业 AI 系统后的数小时内，凸显 Claude 在关键军事场景中的实际应用能力。尽管具体使用细节未公开，但报道指出 Claude 被用于处理多源情报数据、生成行动简报与模拟应对方案。此举引发关于 AI 军事化伦理与监管的广泛讨论。

6. 🚀 Qwen 3.5 将推四款小型稠密开源模型

标签： 阿里云 Qwen 3.5

据开发者 Justin Lin 与 Casper Hansen 在 X 平台透露（https://x.com/JustinLin610/status/2028072650619380067），阿里云通义千问团队即将发布 Qwen 3.5 系列中的四款小型稠密开源模型。这些模型参数规模预计在 1B 至 7B 之间，采用稠密架构设计，兼顾推理效率与性能表现，适用于边缘设备与低延迟场景。新模型将支持中英双语，并在数学推理、代码生成与多轮对话方面进行专项优化。此举有望进一步推动开源大模型生态发展，降低中小企业与研究人员的使用门槛。
