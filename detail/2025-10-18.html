1. 🔧 Google Gemini API集成Maps Grounding

标签： Google Gemini API Google Maps Grounding

Google宣布其Gemini API现已集成Google Maps Grounding工具，该功能允许开发者将Google Maps的实时地理空间数据与Gemini大模型结合，实现基于真实世界位置信息的智能推理。通过该集成，AI应用可调用地图数据作为上下文，提升导航、位置推荐、物流规划等场景的准确性。该功能已在官方开发者博客发布，适用于所有Gemini API用户。

2. 🚀 豆包开启长期记忆功能众测

标签： 豆包 字节跳动 长期记忆功能

字节跳动旗下AI助手豆包宣布开启长期记忆功能的公开测试，并同步发布新一代模型。该功能使豆包能够跨会话记住用户偏好、历史交互内容，实现更个性化的对话体验。新模型在理解能力、响应速度和记忆容量上均有提升，支持更复杂的上下文推理。众测已在小红书平台上线，用户可通过指定链接参与体验。

3. 🔧 Cerebras发布REAP MoE压缩方法

标签： Cerebras REAP MoE

Cerebras发布名为REAP（Retrieval-Enhanced Activation Pruning）的新型MoE（Mixture of Experts）模型压缩技术，旨在降低大规模稀疏模型推理成本。该方法通过动态激活剪枝与检索增强机制，在保持模型性能的同时显著减少计算开销。Cerebras同步开源了基于Qwen3-Coder的REAP压缩模型Qwen3-Coder-REAP-363B-A35B-FP8，采用FP8精度，适用于代码生成与推理任务。

🔗 相关链接：
- https://arxiv.org/abs/2510.13999
- https://huggingface.co/cerebras/Qwen3-Coder-REAP-363B-A35B-FP8

4. 🔧 美团LongCat开源音频编解码器

标签： 美团 LongCat LongCat-Audio-Codec

美团LongCat团队宣布开源其自研音频编解码器LongCat-Audio-Codec，支持高质量语音压缩与重建。该编解码器专为AI语音交互场景设计，具备低延迟、高保真、小体积等特点，适用于语音助手、实时通话、语音生成等应用。代码已在GitHub公开，支持多种采样率与比特率配置，便于开发者集成至端侧或云端语音系统。

🔗 相关链接：
- https://github.com/meituan-longcat/LongCat-Audio-Codec

5. 🚀 GitHub Copilot更新GPT-4.1模型

标签： GitHub Copilot GPT-4.1

GitHub Copilot发布十月更新，引入GPT-4.1作为新一代代码补全模型，显著提升代码生成质量与上下文理解能力。更新还包括对多文件上下文支持、错误修复建议增强、以及更精准的API调用建议。该模型在代码安全性、可读性和性能优化方面表现更优，适用于复杂项目开发与自动化编程场景。

6. 🔧 Firecrawl开源Open Agent Builder

标签： Firecrawl Open Agent Builder AI代理

Firecrawl团队宣布开源Open Agent Builder，一个用于构建和部署AI代理的开发框架。该工具支持可视化流程编排、多模型集成、工具调用与状态管理，旨在降低AI代理开发门槛。开发者可通过拖拽方式构建复杂代理逻辑，并一键部署至云端或边缘设备。项目已在GitHub发布，支持Python与JavaScript。

7. 🚀 李飞飞团队发布RTFM世界模型

标签： 李飞飞 RTFM World Labs

李飞飞领导的World Labs团队发布RTFM（Real-Time Generative World Model），一个支持实时生成与交互的3D世界模型。RTFM能够根据文本或语音指令动态生成可交互的虚拟环境，支持物理模拟、物体交互与多用户协作。模型已在rtfm.worldlabs.ai上线，适用于游戏开发、虚拟训练、数字孪生等场景。

🔗 相关链接：

8. 🚀 DiffSynth发布Face-to-Photo模型

标签： DiffSynth-Studio Face-to-Photo Merjic_AI

DiffSynth-Studio与Merjic_AI联合发布Face-to-Photo模型，可将手绘人脸草图或低质量人脸图像转换为高清真实照片。该模型基于扩散架构，支持细节增强、光照校正与纹理生成，适用于影视制作、游戏角色设计、身份重建等应用。模型已在ModelScope开源，支持本地部署与API调用。

10. 🚀 OpenAI推出企业版MCP支持

标签： OpenAI ChatGPT MCP

OpenAI宣布在ChatGPT企业版中全面支持MCP（Model Context Protocol）连接器，并推出Developer Mode。MCP允许企业将内部系统（如数据库、CRM、ERP）安全接入ChatGPT，实现数据驱动的智能决策。Developer Mode提供高级调试、API调用与权限控制，适用于企业级AI应用开发。功能目前处于Beta阶段。

11. 🔧 xAI API支持服务端工具调用

12. 📈 Nvidia中国市场份额归零

14. 📰 宾大研究称粗鲁提示提升ChatGPT准确率

标签： 宾大 ChatGPT 提示工程

宾夕法尼亚大学研究发现，使用粗鲁或命令式语言提示（如'你必须回答正确'）可显著提高ChatGPT在复杂任务中的准确率。实验显示，在数学推理、事实核查等任务中，粗鲁提示平均提升准确率5-8%。研究人员认为，这可能与模型对指令的优先级判断机制有关，但强调应谨慎使用以避免伦理风险。
