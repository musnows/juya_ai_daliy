1. 🚀 Anthropic 发布 Claude Sonnet 4.6，增强动态网页搜索能力

标签： Anthropic Claude Sonnet 4.6

Anthropic 于2026年2月18日正式发布 Claude Sonnet 4.6 版本。该版本重点优化了网页搜索功能，引入动态过滤机制，显著提升信息检索的准确性与相关性。根据官方博客介绍，新模型在处理实时网络内容时具备更强的上下文理解能力，能够根据用户查询动态调整搜索策略，减少无关结果干扰。此次更新进一步巩固了 Claude 系列在长文本理解和复杂推理任务中的领先地位，尤其适用于研究、内容创作和数据分析等专业场景。

2. 🚀 xAI 上线 Grok 4.20 测试版，马斯克确认发布

标签： xAI Grok 4.20

xAI 于2026年2月18日推出 Grok 4.20 测试版本，埃隆·马斯克在 X 平台（原 Twitter）上亲自确认该版本上线。尽管官方尚未披露具体技术细节，但版本号从常规迭代跃升至 4.20 暗示此次更新可能包含重大架构改进或功能扩展。Grok 系列一贯强调实时信息获取与社交媒体语境理解能力，预计新版本将进一步强化其在动态内容处理和对话交互方面的表现，为后续正式商用版本奠定基础。

3. 🔧 NotebookLM 新增幻灯片Prompt修订与PPTX导出功能

标签： NotebookLM Google

Google 旗下 NotebookLM 于2月18日宣布推出两项重要更新：支持对幻灯片生成过程中的 Prompt 进行精细化修订，并新增直接导出为 PowerPoint（PPTX）格式的功能。这一改进极大提升了用户将 AI 生成的内容转化为正式演示文稿的效率，尤其适用于教育、培训和商业汇报场景。用户现在可以更灵活地控制内容结构、语气与重点，实现从笔记到成品演示的无缝衔接。

4. 🔧 蚂蚁集团开源 Ming-omni-tts 音频生成模型

标签： 蚂蚁集团 Ming-omni-tts

蚂蚁集团于2月18日开源其最新音频生成模型 Ming-omni-tts，提供完整的模型权重、推理代码及演示接口。该模型支持多语言语音合成，具备高保真音质与自然语调表达能力。项目已在 GitHub、ModelScope 等平台同步发布，并附有在线演示 demo，便于开发者快速集成与测试。此举标志着蚂蚁在生成式语音技术领域迈出开源关键一步，有望推动中文及多语种 TTS 应用的普及与发展。

5. 🚀 Cohere Labs 发布 Tiny Aya 多语言轻量模型

标签： Cohere Labs Tiny Aya

Cohere Labs 于2月18日发布 Tiny Aya 多语言模型，主打轻量化与高效推理。该模型在保持多语言理解能力的同时，显著降低计算资源需求，适合部署在边缘设备或低延迟场景中。技术报告指出，Tiny Aya 在多个低资源语言任务上表现优异，填补了当前主流大模型在资源受限环境下的应用空白。模型已同步开源至 Hugging Face，供研究社区使用。

6. 🔧 字节跳动开源 BitDance 多模态模型

标签： 字节跳动 BitDance

字节跳动研究团队于2月18日开源多模态模型 BitDance，支持图像、文本与音频的联合理解与生成。项目代码、文档及预训练模型权重均已发布在 GitHub 和 Hugging Face 平台。据官方介绍，BitDance 在跨模态对齐任务中表现突出，可用于内容审核、智能推荐与多媒体创作等场景。该模型的开放将进一步促进多模态 AI 技术的社区协作与产业落地。

7. 🔧 Cursor 2.5 版本更新，推出插件市场

标签： Cursor 插件市场

AI 编程助手 Cursor 于2月18日发布 2.5 版本重大更新，核心亮点是正式推出插件市场（Marketplace）。开发者可通过该平台安装第三方扩展，增强代码补全、调试与项目管理等功能。此次更新标志着 Cursor 从单一工具向可扩展开发生态演进，有望提升其在专业开发者群体中的竞争力。更新日志显示，新版本还优化了上下文感知能力与响应速度。

8. 🔧 OpenAI 修复 GPT-5.3-Codex 请求重定向漏洞

标签： OpenAI GPT-5.3-Codex

OpenAI 于2月18日确认并修复了 GPT-5.3-Codex 模型中存在的请求重定向安全问题。该漏洞可能导致恶意构造的输入触发非预期行为，影响系统稳定性与数据完整性。官方已在开发者文档中更新相关安全说明，并建议所有集成方升级至最新 API 版本。此次修复体现了 OpenAI 对模型安全性的持续关注，尤其在代码生成类高风险应用中强化防护机制。

9. 📈 Cerebras 下调部分免费层级推理额度

标签： Cerebras

Cerebras 于2月18日调整其推理服务平台的部分免费层级配额，降低免费用户的可用计算资源上限。此次调整涉及入门级免费计划，可能影响个人开发者与小团队的试用体验。公司未明确说明调整原因，但推测与运营成本控制或引导用户转向付费服务有关。用户需关注官方文档以获取最新额度详情。

10. 🔧 Intelligent Internet 开源多Agent协作系统 Common Ground Core

标签： Intelligent Internet Common Ground Core

Intelligent Internet 公司于2月18日开源其多智能体协作框架 Common Ground Core（CGC），旨在支持多个 AI Agent 在共享环境中协同完成任务。该系统提供任务分配、通信协议与状态同步机制，适用于复杂自动化流程与分布式智能应用。项目已在 GitHub 发布，并附有详细技术博客说明其设计理念与应用案例，推动多 Agent 系统研究的开放发展。

11. 📈 Nerve 加入 OpenAI，参与构建 ChatGPT 搜索功能

标签： Nerve OpenAI ChatGPT

AI 初创公司 Nerve 于2月18日宣布正式加入 OpenAI，其团队将参与 ChatGPT 搜索功能的开发工作。Nerve 此前专注于自然语言理解与知识检索技术，此次整合有望加速 ChatGPT 在实时信息获取与精准答案生成方面的能力提升。此举也反映出 OpenAI 持续吸纳外部技术团队以强化核心产品竞争力的战略方向。

12. 📈 传 Moonshot AI 完成7亿美元融资

标签： Moonshot AI

据2月18日消息，中国 AI 公司 Moonshot AI 已完成约7亿美元的新一轮融资。尽管官方尚未正式公告，但多方信源证实该轮融资已接近尾声，资金将主要用于大模型研发、算力基础设施扩建与全球化布局。Moonshot AI 以 Kimi 系列大模型闻名，此次融资规模凸显资本市场对国产基础模型厂商的高度关注与信心。
