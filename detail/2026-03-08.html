1. 📈 智谱向部分用户发放15天订阅费用等值补偿赠金

标签： 智谱 智谱AI

智谱AI近期因服务调整或技术故障，向部分订阅用户发放了为期15天的订阅费用等值补偿赠金。此举旨在弥补用户在服务体验中可能受到的影响，体现了公司对用户权益的重视。补偿以赠金形式发放，用户可在后续订阅中抵扣相应费用。该事件反映出AI服务企业在产品稳定性与用户关系管理方面面临的挑战，同时也展示了通过经济补偿维护用户信任的常见策略。

2. 🚀 腾讯发布视觉语言模型Penguin-VL

标签： 腾讯 Penguin-VL

腾讯正式发布其自研视觉语言模型Penguin-VL，该模型具备多模态理解能力，可同时处理图像与文本信息，支持图像描述生成、视觉问答等任务。作为腾讯在AI多模态领域的重要布局，Penguin-VL有望应用于内容审核、智能客服、教育辅助等场景，提升人机交互的自然性与准确性。

3. 🔧 Claude Code新增/Loop命令支持会话内周期性任务

标签： Claude Code Anthropic

Anthropic推出的Claude Code工具新增/Loop命令功能，允许用户在单一会话中设置周期性执行的任务，提升开发效率与自动化水平。该功能适用于重复性代码检查、定时日志分析等场景，进一步强化了Claude Code作为AI编程助手的实用性，标志着AI辅助开发工具向更智能的交互模式演进。

4. 🚀 xAI上线文本转语音API支持五种语音及情感控制

标签： xAI 文本转语音API

xAI正式推出文本转语音（TTS）API，支持五种不同语音风格，并提供情感控制功能，用户可调节语音的情绪表达，如喜悦、严肃或平静。该API适用于有声内容生成、虚拟助手、无障碍服务等领域，提升了语音合成的自然度与表现力，是xAI在生成式AI应用层的重要拓展。

5. 🔧 QQ支持接入OpenClaw

标签： QQ OpenClaw

腾讯QQ已支持接入OpenClaw，用户可通过QQ平台调用该AI能力，实现更智能的聊天辅助、内容生成或信息检索功能。此次集成进一步拓展了QQ作为社交平台的AI服务能力，也表明OpenClaw在第三方应用中的兼容性与实用性得到验证。

6. 🔧 阿里巴巴与武汉大学联合推出AgeMem记忆框架

标签： 阿里巴巴 武汉大学 AgeMem

阿里巴巴与武汉大学合作研发并推出AgeMem记忆框架，该框架专注于增强AI系统的长期记忆能力，支持信息的高效存储、检索与关联推理。AgeMem有望应用于个性化推荐、智能对话系统与知识管理等领域，推动AI从短期交互向持续学习演进。

7. 📈 Cursor加码自研模型

标签： Cursor

AI编程工具Cursor宣布加大在自研模型上的投入，旨在减少对外部大模型的依赖，提升响应速度、隐私安全性与定制化能力。此举标志着垂直领域AI工具正逐步走向技术自主，通过自研模型优化开发体验，增强产品竞争力。
