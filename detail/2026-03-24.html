1. 🚀 Claude 桌面端上线电脑操控功能，限 macOS 系统

标签： Claude Anthropic macOS

Anthropic 公司推出的 AI 助手 Claude 在其桌面端应用中正式上线电脑操控功能，目前仅支持 macOS 系统。该功能允许用户通过自然语言指令直接控制电脑操作，如打开应用、编辑文档、浏览网页等，标志着 Claude 从被动响应向主动交互的重要演进。此举显著提升了 AI 助手的实用性和自动化能力，可能推动更多 AI 产品向系统级集成发展。该功能于2026年3月24日正式发布，是本期 AI 早报中最受关注的更新。

2. 🔧 OpenClaw 连更两个版本，功能大幅增强

标签： OpenClaw

开源 AI 项目 OpenClaw 连续发布两个新版本，新增大量功能模块，涵盖多模态理解、任务编排与本地执行能力。此次更新显著提升了其在自动化工作流和智能代理（Agent）场景中的表现，支持更复杂的指令解析与跨平台操作。频繁的版本迭代反映出社区活跃度和开发进度的加速，可能为开发者提供更灵活的工具链支持。

3. 🚀 Luma AI 发布 Uni-1 模型，实现推理与视觉同步处理

标签： Luma AI Uni-1

Luma AI 正式推出新一代多模态模型 Uni-1，该模型具备推理与视觉信息同步处理能力，能够在分析图像的同时进行逻辑推理，适用于视频理解、实时场景解析等复杂任务。Uni-1 的发布标志着视觉-语言融合技术的进一步成熟，有望在自动驾驶、智能监控和增强现实等领域发挥重要作用。

4. 🚀 OpenAI 推出 ChatGPT Library 功能，面向付费用户推送

标签： OpenAI ChatGPT Library

OpenAI 向 ChatGPT 付费用户推送全新 Library 功能，允许用户集中管理对话历史、自定义提示模板与常用插件。该功能提升了用户的内容组织效率与个性化体验，进一步强化 ChatGPT 作为生产力工具的定位。目前该功能处于逐步推送阶段，预计将覆盖所有 Plus 和 Team 用户。

5. 🔧 网易云音乐推出 CLI 工具，集成 OpenClaw 实现自动化

标签： 网易云音乐 OpenClaw CLI 工具

网易云音乐发布命令行界面（CLI）工具，并接入 OpenClaw 框架，支持通过脚本或自然语言指令实现音乐播放、歌单管理与用户数据查询等操作。该工具面向开发者和高级用户，提升了音乐服务的可编程性与自动化能力，体现了 AI 与娱乐场景的深度融合。

6. 🚀 千问上线打车能力，支持自然语言叫车与支付

标签： 千问 阿里云

阿里云旗下大模型千问（Qwen）新增打车功能，用户可通过自然语言指令完成叫车、路线选择及支付全流程。该功能整合了出行平台接口与支付系统，实现了端到端的智能出行服务，展示了大模型在生活服务场景中的落地潜力。

7. 🔧 Cursor 介绍新型文本索引技术，降低 Agent 搜索延迟

标签： Cursor 文本索引

AI 编程工具 Cursor 发布技术文章，介绍其研发的新型文本索引机制，通过优化向量检索与上下文缓存策略，显著降低智能代理（Agent）在代码库中搜索相关内容的延迟。该技术提升了代码补全与重构建议的响应速度，有助于改善开发者体验，推动 AI 编程助手的性能边界。
