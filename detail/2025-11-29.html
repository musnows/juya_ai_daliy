1. 🚀 LongCat上线AI生图功能

标签： LongCat LongCat Image

美团LongCat推出AI生图功能，基于LongCat Image模型，主打快速响应、摄影级质感和精准的中文渲染。功能包括自然语言编辑图片，用户可通过描述调整图像内容。该功能已在APP和网站同步上线，支持多种场景下的图像生成需求，提升了中文语境下的AI生图体验。

🔗 相关链接：
- https://mp.weixin.qq.com/s/kBeUr5phfI1wxD9jbH0-lA

2. 🔧 腾讯混元发布3D Studio 1.1及PolyGen 1.5

标签： 腾讯混元 3D Studio PolyGen

腾讯混元发布3D Studio 1.1版本，并接入PolyGen 1.5模型。新版本首次实现端到端原生四边形网格生成，可直接输出具备专业布线结构的3D资产。基础模型升级至混元3D 3.0，建模精度提升约3倍，适用于游戏、影视等行业的3D内容创作，大幅提高建模效率。

🔗 相关链接：
- https://3d.hunyuan.tencent.com/studio
- https://mp.weixin.qq.com/s/QRxLAZZ0JcSJQpOMbAkpqw

3. 🚀 Kimi上线Agentic Slides并开发记忆功能

标签： Kimi Agentic Slides

Kimi国际版上线Agentic Slides功能，支持将PDF、图片等多种文件一键转换为可编辑的PPTX格式幻灯片。功能包括自动排版、内容提取和编辑优化。此外，Kimi正在开发记忆与个性化功能，未来可基于用户历史行为提供定制化服务，提升办公自动化体验。

🔗 相关链接：
- https://x.com/Kimi_Moonshot/status/1994353194420547837

4. 🚀 昆仑万维发布Mureka V7.6和O2音乐模型

标签： 昆仑万维 Mureka Mureka O2

昆仑万维发布Mureka V7.6和Mureka O2音乐生成模型，新模型在音乐性、编曲能力、音质和响应速度上均有显著提升。支持实时音乐生成，适用于音乐创作、游戏配乐等场景，提供更流畅的交互式音乐生成体验，推动AI音乐创作向低延迟、高保真方向发展。

🔗 相关链接：
- https://mp.weixin.qq.com/s/vIKjsFA_oR4-DYBEKDjNSA

5. 🔧 快手开源Keye-VL-671B-A37B多模态模型

标签： 快手 Keye-VL-671B-A37B DeepSeek V3

快手开源6710亿参数的多模态大模型Keye-VL-671B-A37B，基于DeepSeek V3架构打造。模型在文本、视觉、视频理解及数学推理方面表现顶级，支持复杂跨模态任务，如图文问答、视频分析等，为科研与工业界提供高性能开源基础模型。

🔗 相关链接：
- https://huggingface.co/Kwai-Keye/Keye-VL-671B-A37B

6. 🚀 Grok Imagine推出文本到视频功能

标签： Grok Imagine 马斯克

Grok Imagine新增文本到视频生成功能，用户可通过文本提示直接创建视频。马斯克表示，支持音频和10秒时长的升级版功能有望在本周末上线。该功能拓展了AI内容生成边界，适用于短视频创作、广告预览等场景，提升内容生产效率。

🔗 相关链接：
- https://x.com/elonmusk/status/1994221082681733232

7. 📈 Kilo AI发布三大编码模型基准测试

标签： Kilo AI GPT-5.1 Gemini 3.0 Claude Opus 4.5

Kilo AI发布报告，对比GPT-5.1、Gemini 3.0和Claude Opus 4.5三款顶级编码模型。结果显示Claude Opus 4.5综合表现最佳但成本最高；GPT-5.1擅长防御性编码；Gemini 3.0成本最低且能严格遵循指令。报告为开发者选型提供重要参考。

🔗 相关链接：
- https://blog.kilo.ai/p/benchmarking-gpt-51-vs-gemini-30-vs-opus-45

8. 📈 北京推进太空数据中心建设

标签： 北京 太空数据中心

北京正推进太空数据中心建设，规划在700~800km轨道部署大型数据中心系统，分三阶段实施，计划到2035年建成大规模太空数据中心。旨在应对AI带来的算力需求，探索太空边缘计算、低延迟通信等前沿技术，推动航天与AI融合创新。
