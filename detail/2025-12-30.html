1. 🚀 阿里通义开源GUI智能体MAI-UI模型

标签： 阿里巴巴 MAI-UI 通义实验室

阿里巴巴通义实验室于12月30日正式开源MAI-UI系列GUI智能体模型，包括2B和8B两个版本，原生支持主动交互与MCP（模型控制协议）调用，采用端云协同架构设计，可在本地与云端协同执行复杂界面操作任务。该模型能够理解并操作图形用户界面，适用于自动化办公、应用控制和多模态交互场景。目前模型已在Hugging Face平台开源，并提供GitHub代码库与相关论文链接，便于开发者集成与二次开发。此次开源标志着大模型在GUI智能体方向的重要突破，有望推动人机交互范式的演进。

🔗 相关链接：
- https://arxiv.org/abs/2512.22047
- https://huggingface.co/Tongyi-MAI/MAI-UI-2B
- https://huggingface.co/Tongyi-MAI/MAI-UI-8B

2. 🚀 快手KwaiKAT推出Coding Plan订阅包

标签： 快手 KwaiKAT Cat Code PRO v1

3. 🔧 Antigravity回应Agent报错配额问题

标签： Antigravity

针对用户反馈的Agent错误率上升问题，Antigravity官方表示该现象由触发全球配额限制所致，目前已大幅提高系统配额上限。此外，根据官方回复用户评论的信息，未来Antigravity或将为WorkSpace企业账户提供额外使用额度，以提升高负载场景下的服务稳定性。

5. 🚀 Naver开源HyperCLOVA X SEED系列模型

标签： Naver HyperCLOVA X SEED-Omni

韩国Naver公司发布开源模型系列HyperCLOVA X SEED，包含专注于推理的32B视觉语言模型SEED-Thinking，以及支持文本、图像、语音多模态理解与生成的8B全能模型SEED-Omni。该系列配套提供生产级推理系统OmniServe，支持高效部署与大规模应用，适用于智能客服、内容生成与多模态交互等场景。

6. 🚀 fal开源FLUX.2-dev-Turbo图像生成模型

标签： fal FLUX.2-dev-Turbo

7. 🚀 腾讯开源WeDLM-8B-Instruct扩散语言模型

标签： 腾讯 WeDLM-8B-Instruct
