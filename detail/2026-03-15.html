1. 📈 Claude 推出限时促销，非高峰及周末使用额度翻倍

标签： Claude Anthropic

Anthropic 公司旗下 AI 助手 Claude 于2026年3月15日宣布推出限时促销活动，针对非高峰时段及周末的用户使用额度进行翻倍激励。此举旨在优化服务器资源分配，鼓励用户在低峰期使用服务，从而提升整体系统效率与用户体验。该促销未明确结束时间，但强调为‘限时’性质，可能持续数周。此次调整不涉及模型性能或功能变更，仅针对调用配额策略。对于高频使用 Claude 的企业和个人开发者而言，此举可显著降低单位使用成本，尤其在周末或非工作时间进行批量任务处理时更具性价比。该策略也反映出 AI 服务商在资源调度与用户增长之间的平衡考量。

2. 🔧 智谱推迟 GLM Coding Plan 续订与回退窗口

标签： 智谱 AI GLM Coding Plan

智谱 AI 宣布推迟 GLM Coding Plan 的续订与回退机制窗口，原定计划的时间节点被延后，具体新时间表尚未公布。GLM Coding Plan 是智谱面向开发者推出的代码生成与辅助编程服务套餐，支持 GLM 系列大模型在编程场景下的深度调用。此次推迟可能涉及后台系统升级、计费逻辑调整或用户反馈整合。对于已订阅用户而言，回退选项的延迟可能影响其服务切换灵活性，但官方未说明是否影响现有功能使用。该变动反映出大模型服务在商业化进程中仍需优化订阅机制的稳定性与用户体验。

3. 🚀 StepFun 发布 Step 3.5 Flash 模型训练数据

标签： StepFun Step 3.5 Flash

StepFun 公司正式发布 Step 3.5 Flash 模型的训练数据集，公开部分数据来源与构建方法，强调其在轻量化与响应速度上的优化设计。该模型定位为高性价比推理模型，适用于移动端与边缘计算场景。公布的训练数据涵盖多语言文本、代码片段及结构化指令，采用去标识化处理并符合数据合规标准。此举有助于提升模型透明度，增强开发者信任，并为第三方研究提供参考。Step 3.5 Flash 预计将在低延迟应用场景中发挥重要作用，如实时对话、嵌入式 AI 助手等。

4. 🔧 Z Code 发布 V0.22.1，上线浅色主题并接入 OpenCode

标签： Z Code OpenCode

Z Code 于2026年3月15日发布版本 V0.22.1，主要更新包括正式上线浅色主题界面，并正式接入 OpenCode 代码协作平台。此次更新提升了 IDE 的视觉适配性，满足用户在明亮环境下的编码需求。接入 OpenCode 后，用户可直接在 Z Code 中访问开源项目、提交代码变更并参与协作评审，实现开发环境的一体化整合。该版本还优化了语法高亮与自动补全性能，支持更多编程语言。此次更新标志着 Z Code 向全功能开发平台的进一步演进。

5. 📈 消息称字节跳动暂停 Seedance 2.0 全球发布，涉迪士尼版权纠纷

标签： 字节跳动 Seedance 2.0

据内部消息，字节跳动已暂停 AI 视频生成产品 Seedance 2.0 的全球发布计划，原因涉及与迪士尼公司的版权纠纷。Seedance 2.0 是一款基于大模型的短视频自动生成工具，支持角色动作与场景合成。纠纷焦点可能集中在生成内容中未经授权使用迪士尼旗下角色形象或风格模仿。尽管产品已完成内测并准备上线，但法律风险导致发布进程中断。字节跳动尚未官方回应，但已推迟原定于2026年第二季度的全球推广活动。该事件凸显 AI 生成内容在版权合规方面面临的挑战。

6. 🔧 OpenClaw 发布更新，支持接入实时 Chrome 会话

标签： OpenClaw

OpenClaw 发布最新版本，新增支持接入实时 Chrome 浏览器会话功能，允许 AI 代理直接操作网页内容并执行自动化任务。该功能基于浏览器自动化协议实现，用户可通过自然语言指令控制页面跳转、表单填写、数据抓取等操作。更新后，OpenClaw 可应用于网页测试、信息聚合、智能客服等场景，显著提升自动化效率。安全机制方面，系统默认启用沙箱环境，防止恶意操作。此次更新标志着 OpenClaw 从静态分析工具向动态交互平台的转型。

7. 📈 腾讯云启动“龙虾”全国免费安装，40 天覆盖 17 城

标签： 腾讯云 龙虾计划

腾讯云宣布启动代号为“龙虾”的全国免费安装计划，将在40天内覆盖17个主要城市，为中小企业提供免费 AI 基础设施部署服务。该计划涵盖云服务器、数据库、AI 推理节点等资源的免费配额，旨在降低企业上云门槛。覆盖城市包括北京、上海、广州、深圳、成都、杭州等一线及新一线城市。腾讯云表示，此举是其2026年“普惠 AI”战略的重要组成部分，预计将服务超过10万家企业。免费期结束后，用户可选择按需付费或续订优惠套餐。
