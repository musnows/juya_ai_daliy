1. 🚀 MiniMax 发布 M2.7 模型，SWE-Pro 得分 56.22%

标签： MiniMax M2.7 模型 SWE-Pro

MiniMax 正式推出其新一代大模型 M2.7，该模型具备自我进化能力，在 SWE-Pro 基准测试中取得了 56.22% 的得分，显示出在软件工程任务上的显著进步。M2.7 的发布标志着 MiniMax 在提升模型自主优化与复杂任务处理能力方面迈出关键一步，有望在代码生成、系统维护等实际应用场景中发挥更大作用。

2. 🚀 MiniMax 推出 Token Plan，支持全模态统一访问

标签： MiniMax Token Plan

MiniMax 同步发布 Token Plan，旨在实现全模态模型（文本、图像、音频等）的统一访问与管理。该计划通过统一的 token 计费与接口体系，降低开发者调用多模态 AI 服务的复杂度，提升资源调度效率，为构建跨模态 AI 应用提供更灵活的技术支持。

3. 🚀 小米发布 MiMo 系列三款模型并开放 API

标签： 小米 MiMo 系列模型

小米公司正式发布 MiMo 系列三款大模型，涵盖不同规模与能力层级，并同步开放 API 服务。此举标志着小米正式进入大模型商业化服务领域，为开发者与企业用户提供可集成的 AI 能力，可能应用于智能终端、物联网及内容生成等场景。

4. 🔧 Midjourney V8 模型开启早期测试，生成速度提升五倍

标签： Midjourney V8 模型

Midjourney 启动 V8 模型的早期测试，新版本在图像生成速度上实现五倍提升，显著优化了用户体验与创作效率。V8 模型在保持高质量输出的同时，大幅缩短响应时间，有望进一步巩固其在 AI 图像生成领域的领先地位。

5. 🚀 英伟达发布 Nemotron 3 Nano 4B 混合架构模型

标签： 英伟达 Nemotron 3 Nano 4B

英伟达推出 Nemotron 3 Nano 4B 模型，采用混合架构设计，兼顾性能与效率。该模型面向轻量级推理场景，适用于边缘设备与低延迟应用，体现了英伟达在模型优化与部署灵活性方面的持续投入。

6. 🚀 百度千帆发布 40 亿参数 Qianfan-OCR 模型

标签： 百度千帆 Qianfan-OCR

百度千帆平台发布 Qianfan-OCR 模型，参数量为 40 亿，专注于高精度光学字符识别任务。该模型在复杂文档、多语言及低质量图像场景下表现优异，可广泛应用于金融、政务、教育等行业的自动化信息提取。

7. 🚀 Datalab 发布 Chandra OCR 2，olmocr 基准得分 85.9

标签： Datalab Chandra OCR 2 olmocr

Datalab 推出 Chandra OCR 2 模型，在 olmocr 基准测试中获得 85.9 分，刷新 OCR 性能记录。该模型在文本检测与识别精度上实现突破，适用于高精度文档数字化与多语言文本处理场景。
