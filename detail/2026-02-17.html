1. 🚀 阿里千问开源Qwen3.5-397B-A17B大模型

标签： 阿里千问 Qwen3.5-397B-A17B 阿里云百灵

阿里巴巴旗下千问团队于2026年2月17日正式发布并开源Qwen3.5-397B-A17B模型。该模型拥有3970亿参数，采用混合专家架构（MoE），激活参数为170亿，在保持高效推理的同时显著提升模型能力。作为Qwen3.5系列的重要成员，该模型在数学、代码、多语言理解和复杂推理任务上表现突出，支持长上下文处理，适用于企业级AI应用和科研场景。模型已在GitHub、Hugging Face及阿里云百灵平台同步开源，并提供API调用服务，标志着阿里在大模型开源生态建设上的进一步推进。

2. 🚀 蚂蚁百灵发布Ling-2.5-1T大模型

标签： 蚂蚁百灵 Ling-2.5-1T ModelScope

蚂蚁集团旗下的百灵团队推出新一代大模型Ling-2.5-1T，参数量达1万亿，具备更强的语义理解与生成能力。该模型在金融、政务、客服等垂直领域进行了深度优化，支持多轮对话与复杂任务处理，已在ModelScope平台开放试用。结合蚂蚁集团在AI安全与大模型工程化方面的积累，Ling-2.5-1T有望在智能风控、智能投顾等场景中发挥重要作用。

3. 🚀 Manus推出Manus Agents智能体平台

标签： Manus Manus Agents Telegram

AI智能体公司Manus正式发布Manus Agents平台，支持用户通过自然语言指令创建和管理自主运行的AI代理。该平台集成任务规划、工具调用与多模态交互能力，已在Telegram上线测试版本。Manus Agents旨在降低AI智能体开发门槛，推动个人与企业级自动化应用落地，被视为通用人工智能（AGI）路径上的重要探索。

4. 🔧 Kilo上线优化版Grok Code Fast 1

标签： Kilo Grok Code Fast 1

AI编程工具公司Kilo发布Grok Code Fast 1的优化版本，显著提升代码生成速度与准确性。新版本在Python、JavaScript等主流语言支持上表现更佳，支持实时代码补全与错误修复，适用于开发者日常编程场景。该版本已在Kilo官方博客公布，并面向企业用户提供定制化部署方案。

5. 📈 智谱启动合伙人计划应对GLM-5算力瓶颈

标签： 智谱AI GLM-5 合伙人计划

智谱AI宣布启动合伙人计划，旨在联合行业伙伴共同解决GLM-5大模型训练与部署中的算力资源瓶颈。该计划将整合云计算、芯片与数据中心资源，构建分布式训练生态，以支撑更大规模模型的迭代。此举反映当前大模型发展对基础设施的高度依赖，也体现智谱在生态协同方面的战略布局。

6. 📈 Flapping Airplanes获1.8亿美元融资

标签： Flapping Airplanes AI架构创新

AI初创公司Flapping Airplanes宣布完成1.8亿美元融资，由多家顶级风投机构领投。该公司专注于探索非传统AI架构，致力于开发“真正激进不同”的AI系统，可能涉及类脑计算或新型神经网络范式。资金将用于技术研发与团队扩张，其创新方向或将对AI基础理论产生深远影响。

7. 📈 Ricursive Intelligence完成3.35亿美元融资

标签： Ricursive Intelligence AI芯片

AI芯片初创公司Ricursive Intelligence宣布完成3.35亿美元融资，估值达40亿美元。该公司专注于高性能、低功耗AI推理芯片研发，产品适用于边缘计算与数据中心场景。此次融资将加速其芯片量产与商业化进程，有望在AI算力竞争中占据一席之地。

8. 📈 传Kimi K3计划于3月发布

标签： Kimi K3 月之暗面

据社交媒体消息，月之暗面（Moonshot AI）旗下Kimi系列大模型K3计划于2026年3月正式发布。尽管官方尚未确认，但业内推测K3将在上下文长度、多模态能力与推理性能上实现重大突破。若属实，K3将进一步加剧国内大模型市场竞争。
