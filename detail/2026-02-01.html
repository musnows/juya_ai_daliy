1. 🚀 Gemini 测试从 ChatGPT 导入对话历史功能

标签： Gemini ChatGPT Google

谷歌旗下 AI 助手 Gemini 正在测试一项新功能，允许用户从 ChatGPT 等平台导入历史对话记录。该功能旨在提升用户体验的连续性，使用户在不同 AI 平台间迁移时能保留重要对话上下文。此举可能打破平台间数据孤岛，增强 Gemini 在多轮对话和个性化服务方面的竞争力。目前该功能处于测试阶段，具体上线时间尚未公布。

2. 🔧 LM Studio 更新支持本地运行 Claude Code

标签： LM Studio Claude Code Anthropic

本地 AI 模型运行平台 LM Studio 发布更新，新增对 Anthropic 的 Claude Code 模型的支持，用户可在本地设备上直接运行该模型，无需依赖云端服务。这一更新提升了数据隐私性和响应速度，适用于对安全性和离线能力有较高要求的开发者和企业用户。

3. 🚀 V0 新增导入 GitHub 项目功能

标签： V0 GitHub

AI 编程工具 V0 推出新功能，支持用户直接导入 GitHub 项目代码库，实现快速项目初始化与代码生成。该功能可显著提升开发效率，尤其适用于前端项目搭建和原型开发，进一步巩固 V0 在 AI 辅助编程领域的领先地位。

4. 📈 英伟达据称搁置与 OpenAI 的千亿协议

标签： Nvidia OpenAI

据消息披露，英伟达已搁置与 OpenAI 达成的价值千亿美元的合作协议。该协议原计划涉及大规模 AI 算力基础设施共建，搁置原因尚不明确，可能影响双方在高端芯片与模型训练方面的长期战略协同。

5. 📈 苹果 AI 核心团队动荡，多人转投 Meta 与谷歌

标签： Apple Meta Google

苹果公司人工智能核心团队近期出现人员流失，多名关键成员已跳槽至 Meta 和谷歌等竞争对手公司。这一变动可能影响苹果在生成式 AI 和大型语言模型领域的研发进度，引发外界对其 AI 战略执行力的担忧。

6. 📈 SpaceX 申请部署百万卫星轨道数据中心

标签： SpaceX Starlink

SpaceX 向监管机构提交申请，计划部署百万颗卫星构建轨道数据中心网络，旨在提供全球覆盖的低延迟云计算与 AI 算力服务。该项目若实施，将极大拓展太空基础设施的应用边界，可能重塑全球数据服务格局。

7. 🔧 nanochat 以百美元内成本训练出 GPT-2 级别模型

标签： nanochat GPT-2

开源项目 nanochat 成功在百美元以内的成本下训练出性能接近 GPT-2 的轻量级语言模型。该项目采用高效训练策略与优化算法，显著降低 AI 模型开发门槛，为资源有限的个人开发者和小型团队提供了可行方案。

8. 🚀 快手可灵 AI 预告 Kling 3.0 全能视频模型

标签： 可灵 AI Kling 3.0 快手

快手旗下 AI 视频生成平台可灵 AI 发布预告，即将推出 Kling 3.0 版本，宣称具备“全能视频生成”能力，支持更高分辨率、更长时长与更复杂场景的视频创作。该模型有望进一步推动 AI 视频生成技术的商业化应用。
