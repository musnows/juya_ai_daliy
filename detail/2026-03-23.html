1. 🚀 微信正式推出 ClawBot 插件，支持接入 OpenClaw

标签： 微信 ClawBot OpenClaw

微信于2026年3月23日正式推出名为 ClawBot 的插件，该插件支持接入 OpenClaw 平台，为用户提供更智能的自动化交互能力。ClawBot 的发布标志着微信在 AI 插件生态上的进一步拓展，用户可通过该插件实现更高效的对话管理与任务执行。结合 OpenClaw 的技术支持，ClawBot 可能具备自然语言理解与多轮对话能力，适用于客服、信息检索等场景。此次更新是微信在 AI 助手领域的重要布局，有望提升用户在复杂任务中的操作效率。

2. 🔧 Claude Code 测试新版 /init，引入交互式配置引导

标签： Claude Code

Claude Code 正在测试新版 /init 命令，该版本引入了交互式配置引导功能，帮助用户更便捷地完成初始设置与个性化配置。这一更新优化了开发者和高级用户的使用体验，降低了上手门槛。通过逐步引导，用户可快速定义工作流、权限设置与集成环境，提升开发效率。该功能可能面向编程辅助与自动化脚本场景，进一步强化 Claude Code 在 AI 编程工具中的竞争力。

3. 🚀 阶跃星辰上线 Step Plan 订阅套餐，月费最低25元起

标签： 阶跃星辰 Step Plan

阶跃星辰于2026年3月23日正式推出 Step Plan 订阅套餐，月费最低为25元人民币，提供分级服务以满足不同用户需求。该套餐可能涵盖模型调用额度、优先响应、高级功能访问等权益，旨在推动其 AI 服务的商业化落地。Step Plan 的推出标志着阶跃星辰从技术输出向可持续商业模式转型，有助于扩大用户基础并提升服务可及性。

4. 🔧 MiniMax 调整 Token Plan 额度方案，提供多模态模型每日配额

标签： MiniMax Token Plan

MiniMax 对其 Token Plan 额度方案进行调整，新增对多模态模型的每日配额支持，用户可在限定额度内调用图像、文本等多模态 AI 能力。此次调整优化了资源分配机制，提升模型使用灵活性与成本控制能力。该方案适用于内容生成、跨模态理解等应用场景，有助于开发者更高效地集成 MiniMax 的多模态技术。

5. 🔧 MiniMax AI 开源 Skills 仓库，涵盖九大核心技能

标签： MiniMax AI Skills 仓库

MiniMax AI 正式开源其 Skills 仓库，涵盖九大核心技能，包括文本生成、图像理解、语音处理等关键 AI 能力。该仓库为开发者提供可复用的技能模块，支持快速构建定制化 AI 应用。开源举措有助于推动社区协作与技术迭代，降低 AI 应用开发门槛，加速行业创新。

6. 📈 阿里巴巴确认将持续开源 Qwen 及 Wan 新模型

标签： 阿里巴巴 Qwen Wan

阿里巴巴官方确认将持续推进 Qwen 和 Wan 系列模型的开源进程，未来将发布更多新版本并开放模型权重。此举彰显其在开源 AI 生态中的长期承诺，有助于促进大模型技术的普及与研究。开源策略将吸引全球开发者参与模型优化与应用创新，进一步巩固阿里在 AI 领域的影响力。

7. 🔧 MiniMax 确认 M2.7 权重将开源，预计两周内发布

标签： MiniMax M2.7

MiniMax 宣布其 M2.7 模型权重将在两周内正式开源，为研究者和开发者提供完整模型访问权限。该模型可能具备更强的语言理解与生成能力，开源后将推动学术研究与商业应用的深度融合。此举也体现了 MiniMax 在推动 AI 技术民主化方面的积极态度。

8. 🔧 MiniMax 确认 M3 模型将支持视觉功能

标签： MiniMax M3 模型

MiniMax 确认其下一代模型 M3 将支持视觉功能，实现图像识别、分析与生成能力。该模型预计将整合多模态技术，提升在复杂场景下的综合理解能力。M3 的发布将进一步增强 MiniMax 在视觉 AI 领域的竞争力，适用于智能监控、内容创作等应用场景。
