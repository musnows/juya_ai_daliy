1. 🚀 MiniMax上线角色扮演模型M2-her API

标签： MiniMax M2-her 角色扮演模型

AI公司MiniMax于2026年1月25日正式上线其角色扮演模型M2-her的API接口。该模型专为角色驱动对话场景设计，支持开发者通过API调用实现高度拟人化的角色交互体验。用户可通过MiniMax官方平台（https://platform.minimax.io）访问相关文档，包括模型介绍（https://platform.minimax.io/docs/guides/models-intro）和文本聊天API参考（https://platform.minimax.io/docs/api-reference/text-chat）。此次发布标志着MiniMax在垂直领域语言模型应用上的进一步拓展，尤其适用于虚拟偶像、游戏NPC、客服助手等需要长期角色一致性的应用场景。M2-her API的推出将降低开发者集成高质量角色扮演能力的门槛，推动AI角色经济生态发展。

2. 🔧 字节跳动发布扩散代码模型Stable-DiffCoder-8B

标签： 字节跳动 Stable-DiffCoder-8B Seed团队

字节跳动旗下Seed团队于2026年1月25日发布两款扩散代码生成模型：Stable-DiffCoder-8B-Base与Stable-DiffCoder-8B-Instruct，模型参数规模达80亿。Base版本为基础预训练模型，适用于通用代码生成任务；Instruct版本经过指令微调，能更好地理解自然语言指令并生成符合需求的代码。该模型基于扩散架构，区别于传统自回归生成方式，可能在代码结构完整性与逻辑一致性方面具备优势。模型已开源并托管于Hugging Face平台，开发者可免费使用。此举表明字节跳动在AI编程助手领域持续投入，有望提升软件开发效率，推动低代码/无代码生态演进。

3. 🚀 MiroThinker新增报告生成与多格式上传功能

标签： MiroThinker 报告生成 多格式上传

AI工具平台MiroThinker于2026年1月25日推出报告自动生成功能，并支持多种文件格式上传处理。用户可通过上传PDF、Word、PPT等文档，由系统解析内容并生成结构化分析报告。该功能整合了文档理解、信息抽取与文本生成技术，适用于学术研究、商业分析、内容摘要等场景。平台官网（https://mirothinker.io）已更新相关功能入口，标志着MiroThinker从单一问答工具向综合性知识处理平台演进，提升用户在信息整合与输出效率方面的体验。

4. 📈 OpenAI正为ChatGPT开发购物车功能

标签： OpenAI ChatGPT 购物车功能

据社交媒体消息，OpenAI正在为ChatGPT开发内置购物车功能，允许用户在对话过程中直接完成商品选购与结算。该功能可能整合第三方电商平台API，实现从咨询、比价到下单的一站式购物体验。虽然官方尚未正式公布，但此举若落地，将显著拓展ChatGPT的商业应用场景，推动AI助手向消费终端延伸，可能重塑人机交互中的电商服务模式。
