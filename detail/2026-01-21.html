1. 🚀 OpenAI 推出 ChatGPT 年龄预测功能

标签： OpenAI ChatGPT

OpenAI 于2026年1月21日正式在 ChatGPT 中上线年龄预测功能，用户可通过对话交互获取基于语言风格、用词习惯和认知复杂度等多维度分析的年龄区间评估。该功能依托于 ChatGPT 内部大模型对用户输入内容的语义理解与行为建模能力，结合训练数据中的年龄相关语言特征进行推断。尽管 OpenAI 未公开具体算法细节，但强调该功能仅用于趣味互动与用户体验优化，不用于身份验证或隐私收集。此举可能引发关于AI对用户心理与行为推断边界的讨论，同时也展示了大模型在个性化交互中的新应用方向。

2. 🚀 面壁智能发布 AgentCPM-Report 开源模型

标签： 面壁智能 AgentCPM-Report

面壁智能于2026年1月21日发布 AgentCPM-Report 开源模型，专注于自动化报告生成任务。该模型基于 CPM 架构优化，支持多模态输入与结构化输出，适用于金融、科研、市场分析等领域的长文本生成场景。作为开源项目，AgentCPM-Report 提供完整训练代码与预训练权重，鼓励社区参与迭代与定制化开发，有望推动企业级AI报告工具的平民化与普及。

3. 🔧 Overworld 发布 Waypoint-1 实时扩散世界模型

标签： Overworld Waypoint-1

Overworld 公司于2026年1月21日推出 Waypoint-1，全球首个支持实时交互的扩散世界模型。该模型能够在用户输入指令后即时生成动态3D环境或虚拟场景，适用于游戏开发、虚拟现实与元宇宙内容创作。Waypoint-1 采用新型时空扩散架构，实现低延迟、高保真的世界构建，标志着生成式AI从静态图像向动态环境生成的重大突破。

4. 🚀 Liquid AI 推出 LFM2.5-1.2B-Thinking 端侧推理模型

标签： Liquid AI LFM2.5-1.2B-Thinking

Liquid AI 于2026年1月21日发布 LFM2.5-1.2B-Thinking，一款专为端侧设备优化的轻量级推理模型，参数量为1.2B。该模型强调‘思考链’能力，支持复杂逻辑推理与多步决策，适用于手机、嵌入式设备等资源受限环境。LFM2.5-1.2B-Thinking 在保持高性能的同时显著降低能耗，推动AI在边缘计算场景中的落地应用。

5. 🚀 LightOn 发布 OCR 模型 LightOnOCR-2

标签： LightOn LightOnOCR-2

LightOn 公司于2026年1月21日推出新一代OCR模型 LightOnOCR-2，支持多语言文本识别、手写体解析与复杂版式还原。该模型在低光照、模糊图像与倾斜文档等挑战性条件下表现优异，准确率较前代提升18%。LightOnOCR-2 可集成至文档管理系统、自动化办公流程与智能扫描设备中，提升信息数字化效率。

6. 🔧 llama.cpp 集成 Anthropic Messages API

标签： llama.cpp Anthropic

开源项目 llama.cpp 于2026年1月21日宣布集成 Anthropic Messages API，允许开发者在本地环境中调用 Claude 系列模型的对话与推理能力。此举打破了此前仅支持 OpenAI 等少数厂商API的限制，增强了开源生态的兼容性与灵活性，为隐私敏感型应用提供了新的部署选项。

7. 🚀 Anthropic 宣布 Claude Code VS Code 扩展正式上线

标签： Anthropic Claude Code

Anthropic 于2026年1月21日正式发布 Claude Code 的 VS Code 扩展版本，开发者可直接在集成开发环境中调用 Claude 模型进行代码补全、错误修复与文档生成。该扩展支持上下文感知编程建议，提升开发效率，标志着 Claude 生态向开发者工具链的深度渗透。

8. 🚀 Anthropic 推出 Claude 健康数据连接功能 Beta 版

标签： Anthropic Claude

Anthropic 于2026年1月21日推出 Claude 健康数据连接功能的 Beta 版本，允许用户授权连接电子健康记录（EHR）、可穿戴设备数据等，用于个性化健康建议与症状分析。该功能严格遵循HIPAA等隐私标准，数据加密传输且本地处理优先，探索AI在医疗辅助领域的合规应用路径。

9. 🚀 MiniMax 发布 Agent 2.0 及桌面端 AI 工作空间

11. 🔧 OpenRouter 发布 Response Healing 插件自动修复 JSON 格式错误

OpenRouter 于2026年1月21日发布 Response Healing 插件，可自动检测并修复大模型返回的 JSON 格式错误，如缺失引号、括号不匹配等。该插件集成于 API 调用流程中，提升开发者对接AI服务的稳定性与容错能力，减少因格式问题导致的系统异常。
