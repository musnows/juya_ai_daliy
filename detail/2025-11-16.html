1. 🚀 OpenRouter上线Sherlock系列模型

标签： OpenRouter Sherlock系列模型

OpenRouter于近日正式上线Sherlock系列AI模型，该系列模型专注于提升推理与代码生成能力，支持多轮对话与复杂任务处理。此次上线进一步丰富了OpenRouter平台上的模型选择，为开发者提供更高性能的API接入选项。Sherlock系列模型具备低延迟、高并发处理能力，适用于智能客服、自动化编程、数据分析等场景。平台通过统一接口支持多种开源与闭源模型，Sherlock系列的加入增强了其在推理密集型任务中的竞争力。

2. 🔧 GitHub Copilot公测Raptor mini模型

标签： GitHub Copilot Raptor mini

GitHub于2025年11月10日宣布，其AI编程助手GitHub Copilot正式推出Raptor mini模型的公开预览版。Raptor mini是专为轻量化代码补全与上下文理解优化的轻量级模型，旨在提升在低资源环境下的响应速度与准确性。该模型已在Copilot服务中逐步推送，支持主流编程语言，适用于IDE中的实时代码建议、函数生成与错误修复。此次公测标志着GitHub在模型分层策略上的深化，为不同性能需求的开发者提供更灵活的选择。

🔗 相关链接：
- https://github.blog/changelog/2025-11-10-raptor-mini-is-rolling-out-in-public-preview-for-github-copilot/

3. 🔧 iFlow CLI免费开放MiniMax-M2与Kimi-K2-Thinking

标签： iFlow CLI MiniMax-M2 Kimi-K2-Thinking

iFlow CLI宣布免费开放对MiniMax-M2与Kimi-K2-Thinking模型的支持，用户可通过命令行工具直接调用这两款高性能大模型。MiniMax-M2以多模态理解和长文本生成见长，Kimi-K2-Thinking则专注于增强推理能力，支持链式思考与复杂逻辑分析。此次开放降低了开发者接入先进模型的门槛，适用于自动化脚本、知识问答、内容生成等CLI场景，推动终端环境下的AI原生应用发展。

4. 🔧 Gemini CLI发布v0.15.1补丁

标签： Gemini CLI Google Gemini

Gemini CLI于近日发布v0.15.1版本补丁，主要修复了上一版本中的API连接稳定性问题，并优化了本地缓存机制与错误日志输出。该版本还增强了对多账户切换的支持，提升了在Linux与macOS环境下的兼容性。作为Google Gemini生态的命令行工具，Gemini CLI v0.15.1的发布有助于开发者更高效地集成Gemini模型至自动化流程中，适用于脚本化AI调用、批量内容生成与系统监控等场景。

5. 🚀 Factory推出Ultra订阅档

标签： Factory Ultra订阅档

Factory AI宣布推出全新Ultra订阅档位，面向企业级用户提供更高优先级API访问、专属模型微调支持与SLA保障服务。Ultra档包含每月1000万次模型调用、7×24小时技术支持以及定制化推理加速选项，适用于高负载AI应用部署。该档位进一步丰富了Factory的订阅体系，满足金融、医疗、制造等行业对稳定性与性能的高要求，推动其商业化生态的扩展。

6. 🔧 Cerebras发布MiniMax-M2-REAP-162B-A10B稀疏模型

标签： Cerebras MiniMax-M2-REAP-162B-A10B

Cerebras在Hugging Face上发布了MiniMax-M2-REAP-162B-A10B稀疏模型，该模型基于MiniMax-M2架构，采用1620亿参数总量，其中激活参数为100亿（A10B），属于稀疏激活大模型（MoE架构）。REAP（Retrieval-Enhanced Adaptive Processing）技术增强了模型在长文档理解与知识检索任务中的表现。该模型专为Cerebras Wafer-Scale Engine优化，具备高吞吐、低延迟特性，适用于大规模语义搜索、智能问答与科研分析等高性能计算场景。

7. 📈 AMD启动高校春雨计划

标签： AMD 春雨计划

AMD正式启动面向中国高校的‘春雨计划’，旨在通过提供GPU硬件、AI开发工具与课程资源，支持高校在人工智能、高性能计算与开源生态领域的教学与科研。该计划将联合多所重点高校共建AI实验室，推广ROCm开源计算平台，并设立专项奖学金与竞赛项目。‘春雨计划’是AMD深化本地生态合作的重要举措，有望加速国产AI基础设施的人才培养与技术落地。

8. 📈 Similarweb：Gemini占比升至13.7%
