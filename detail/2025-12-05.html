1. 🚀 Google上线Gemini 3 Deep Think模式

标签： Google Gemini 3 Deep Think

谷歌正式为AI Ultra订阅用户推出Gemini 3 Deep Think模式，基于Gemini 3 Pro模型，采用并行推理技术，专注于解决复杂数学、科学和逻辑问题。该模式在多项行业基准测试中表现优异，显著提升了推理效率和准确性。Deep Think模式的推出标志着谷歌在AI推理能力上的进一步突破，尤其适用于需要多步推理和跨领域知识整合的高难度任务，如学术研究、工程建模和高级编程问题求解。该功能现已面向AI Ultra用户开放，是视频标题中重点强调的‘宇宙最强AI’核心内容。

2. 🚀 OpenAI开放GPT-5.1-Codex-Max API访问

标签： OpenAI GPT-5.1-Codex-Max Agent Coding

OpenAI正式开放GPT-5.1-Codex-Max模型的API访问，该模型专为Agent Coding设计，支持复杂代码生成、调试和自动化开发流程。目前，Cursor、GitHub Copilot、Replit等主流开发工具已获授权并正在集成该模型。GPT-5.1-Codex-Max在代码理解、上下文感知和生成质量方面显著优于前代，支持多语言、多框架的端到端开发。该API的开放进一步推动了AI编程的普及，尤其适用于企业级自动化开发场景。

3. 🔧 Cursor优化Agent支持Codex模型并限免

标签： Cursor GPT-5.1-Codex-Max AI Agent

Cursor深度优化其AI Agent，以更好地支持OpenAI GPT-5.1-Codex-Max模型，提升编码性能、响应速度和上下文理解能力。优化后，Agent可更精准地执行代码补全、错误修复和项目重构任务。为推广新集成，Cursor向订阅用户限时免费开放GPT-5.1-Codex-Max模型使用权限，用户可在限定时间内无限制调用该模型进行开发。此举显著降低了高级AI编程工具的使用门槛。

4. 🔧 NotebookLM iOS新增图表与幻灯片功能

标签： NotebookLM 信息图表 幻灯片生成

标签： Anthropic 安斯特尔 访谈数据集

Anthropic推出名为‘安斯特尔’（Anthropic Research Tool）的新型研究工具，可辅助进行访谈设计、执行和结果分析，支持自动提问、情感识别和主题聚类。目前该工具已开启为期一周的公开试点。同时，Anthropic公开发布了利用该工具完成的1250名专业人士的访谈数据集，涵盖AI、医疗、金融等多个领域，为AI社会影响研究提供高质量训练数据。
