1. 🚀 Google推出Gemini 3 Deep Think模式

标签： Google Gemini 3 Deep Think

Google正式为AI Ultra订阅用户推出Gemini 3 Deep Think模式，该模式基于Gemini 3 Pro模型，采用并行推理技术，专注于解决复杂数学、科学和逻辑问题。功能包括多路径推理、长链思考、高精度问题分解，支持在STEM领域实现专家级推理能力。在多项行业基准测试（如MATH、GPQA、HumanEval）中表现超越同类模型，尤其在多步推理和跨学科问题求解上优势显著。该模式通过Google AI Studio和Vertex AI开放调用，适用于科研、教育、工程仿真等高阶推理场景，标志着大模型向深度思考能力迈进一步。

2. 🚀 OpenAI开放GPT-5.1 Code XX Max API

标签： OpenAI GPT-5.1 Code XX Max Cody

OpenAI正式开放GPT-5.1 Code XX Max模型的API访问权限，该模型专为Agent Coding设计，特点是支持多轮代码生成、上下文感知补全、错误自动修复与测试用例生成。模型在CodeXGLUE、HumanEval-X等编码基准测试中刷新SOTA成绩。目前，Cody、GitHub Copilot、Replit等主流开发工具正在集成该模型。Cody已深度优化其Agent架构以适配GPT-5.1 Code XX Max，显著提升代码生成准确率和执行效率。该模型对Cody订阅用户限时免费开放，推动企业级智能编程工具升级。

3. 🔧 Notebook LM新增图表与幻灯片生成

标签： Google Notebook LM 信息图表生成

Google Notebook LM的iOS应用新增信息图表和幻灯片自动生成功能，用户可通过自然语言指令将研究内容转化为可视化图表或完整演示文稿。功能包括图表类型智能推荐、数据自动提取、排版优化与多格式导出。聊天定制功能的字符限制从500扩展至10,000，支持长文档深度交互。该更新显著提升学术、咨询、教育等场景下的知识表达效率，实现从文本分析到成果输出的端到端自动化。

4. 🚀 Anthropic发布新型研究工具AnthroTel

标签： Anthropic AnthroTel 访谈数据集

Anthropic推出名为AnthroTel的新型研究工具，支持自动化访谈执行与结果分析，功能包括语音/文本访谈调度、多轮对话管理、情感识别、主题聚类与报告生成。工具已开启为期一周的公开试点，面向社会科学、市场调研、用户研究等领域。Anthropic同步公开发布基于该工具完成的1,250名专业人士访谈数据集，涵盖AI伦理、工作流变革等主题，为行业提供高质量定性研究资源。

5. 🔧 Anthropic推出Claude 4.5结构化输出功能

标签： Anthropic Claude 4.5 结构化输出

Anthropic为Claude 4.5模型推出结构化输出的Beta功能，开发者可通过Cloud API指定JSON Schema，确保模型响应严格匹配预设格式。特点是支持嵌套结构、类型校验、必填字段控制，适用于API集成、数据抽取、自动化工作流等场景。该功能显著降低后处理成本，提升AI系统与企业系统的兼容性，推动AI在金融、医疗、政务等结构化数据要求高的领域落地。

6. 🚀 HeyGen上线数字人2.0功能

标签： HeyGen 数字人2.0 神经渲染

HeyGen正式上线数字人2.0功能，新版本在表演力、手部动作、口型同步方面实现全面升级，支持生成最长5分钟的高保真视频。功能包括多语言口型驱动、情绪表达控制、背景动态合成与实时渲染。采用端到端神经渲染架构，结合3D姿态估计与语音-视觉对齐技术，显著提升虚拟人自然度。适用于电商直播、教育视频、企业宣传等场景，降低视频制作成本。

7. 🔧 千问APP整合文档与PPT对话入口

标签： 千问APP 文档处理 PPT生成

千问APP发布办公场景功能更新，将文档处理与PPT制作整合至统一对话入口。用户可通过自然语言指令完成文档撰写、格式调整、PPT生成与排版优化。功能包括多文档协同、模板推荐、内容自动摘要与图表插入。支持在手机端实现端到端办公流程，提升移动办公效率，推动大模型在轻量化办公场景的普及。

8. 🚀 微软发布轻量级实时TTS模型WebVoice

微软发布轻量级实时文本转语音模型WebVoice Real-Time 0.5B，支持流式输入，延迟约300ms，生成语音可立即播放。特点是模型仅0.5B参数，适用于边缘设备与低带宽环境，支持多语种、多音色与情感调节。适用于实时语音助手、客服系统、无障碍服务等场景，推动低延迟语音交互普及。
