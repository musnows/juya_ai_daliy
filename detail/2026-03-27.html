1. 🚀 Google 发布 Gemini 3.1 Flash Live 实时音频模型

标签： Google Gemini 3.1 Flash Live

谷歌于2026年3月27日正式发布 Gemini 3.1 Flash Live，这是一款支持实时音频交互的多模态AI模型。该模型支持90种语言的实时对话，具备低延迟响应能力，适用于跨语言客服、实时会议翻译和无障碍沟通等场景。作为Gemini系列的重要更新，Flash Live版本在保持高性能的同时优化了推理效率，进一步扩展了谷歌在实时AI交互领域的技术布局。此次发布也标志着谷歌在语音-语言融合模型方向上的持续投入。

2. 🚀 Suno 推出 Suno v5.5 模型及个性化声音功能

标签： Suno Suno v5.5

Suno 正式推出其最新版本 Suno v5.5 模型，新增个性化声音定制功能，用户可根据需求生成专属音色。该版本在音乐生成质量、节奏控制和歌词同步方面均有显著提升，支持更复杂的编曲结构和多语言歌词创作。此次更新进一步巩固了Suno在AI音乐生成领域的领先地位，为内容创作者和音乐制作人提供了更灵活的工具。

3. 🔧 Cohere 发布开源语音识别模型 Cohere Transcribe

标签： Cohere Cohere Transcribe

Cohere 发布开源语音识别模型 Cohere Transcribe，支持高精度语音转文本功能，适用于会议记录、字幕生成和语音助手等应用场景。该模型采用先进的端到端架构，具备良好的多语言支持和噪声鲁棒性。作为开源项目，Cohere Transcribe 将推动语音识别技术的普及与社区协作发展。

4. 🚀 Mistral AI 发布 Voxtral TTS 语音合成模型

标签： Mistral AI Voxtral TTS

Mistral AI 发布 Voxtral TTS 模型，专注于高质量文本到语音转换。该模型支持自然流畅的语音生成，具备多说话人支持和情感语调控制能力，适用于有声书、虚拟助手和无障碍辅助工具等领域。Voxtral TTS 的发布进一步丰富了 Mistral AI 在语音技术栈中的产品布局。

5. 🔧 OpenDataLab 发布 MinerU-Diffusion-V1 图像生成模型

标签： OpenDataLab MinerU-Diffusion-V1

OpenDataLab 发布 MinerU-Diffusion-V1，一款基于扩散模型的开源图像生成系统。该模型支持高分辨率图像合成，具备良好的语义理解能力和细节还原度，适用于设计辅助、内容创作和科研可视化等场景。作为开源项目，MinerU-Diffusion-V1 为社区提供了可复用的图像生成基础设施。

6. 🔧 Meta 发布 TRIBE v2 三模态人脑响应预测模型

标签： Meta TRIBE v2

Meta 发布 TRIBE v2，一款三模态AI模型，能够预测人类大脑在观看视觉、听觉和语言刺激时的 fMRI 响应。该模型结合神经科学与深度学习，为理解人类认知机制提供了新工具，有望应用于脑机接口、神经反馈治疗和认知研究等领域。

7. 🚀 Chroma 发布 200 亿参数 Context-1 开源搜索模型

标签： Chroma Context-1

Chroma 发布拥有200亿参数的 Context-1 模型，这是一款专为 agentic 搜索设计的开源系统。该模型支持上下文感知的信息检索与推理，能够理解复杂查询意图并生成结构化响应，适用于智能问答、知识管理和自动化决策支持系统。

8. 🔧 英伟达发布 gpt-oss-puzzle-88B 剪枝量化模型

标签： 英伟达 gpt-oss-puzzle-88B

英伟达发布 gpt-oss-puzzle-88B，一款经过剪枝与量化优化的大型语言模型。该模型在保持较高推理性能的同时显著降低了计算资源需求，适用于边缘设备和低成本部署场景，体现了英伟达在模型压缩与高效推理方面的技术积累。

11. 🔧 Claude Code 推出云端 auto-fix 功能支持 PR 自动化修复
